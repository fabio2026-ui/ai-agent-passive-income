## 三、LTV（客户生命周期价值）计算

### 3.1 公式

**LTV = ARPU × (1 / 月流失率)**

或更精确版本：
**LTV = ARPU × 毛利率 × 客户平均寿命**

### 3.2 各项目LTV分析

#### Etsy计算器

| 参数 | 数值 | 说明 |
|-----|------|------|
| ARPU | €9.99/月 | 订阅费用 |
| 毛利率 | 85% | 高毛利SaaS |
| 月流失率 | 5% | SMB SaaS基准（3-7%） |
| 平均客户寿命 | 20个月 | 1/0.05 |
| **LTV** | **€170** | 9.99 × 0.85 × 20 |
| 保守LTV | **€135** | 考虑支付失败等因素 |

| 参数 | 数值 | 说明 |
|-----|------|------|
| ARPU | €40/月 | 加权平均（Starter/Pro/Enterprise） |
| 毛利率 | 80% | API调用成本 |
| 月流失率 | 3% | B2B SaaS基准（1-3%） |
| 平均客户寿命 | 33个月 | 1/0.03 |

#### Chrome扩展

### 5.2 盈亏平衡计算

**公式**：盈亏平衡用户数 = 月固定成本 / (ARPU - 变动成本/用户)

#### Etsy计算器

- 月固定成本：€450
- ARPU：€9.99
- 变动成本/用户：€0.50（支付处理+邮件）
- **盈亏平衡点**：450 / (9.99 - 0.50) = **48个付费用户**

预计达到时间：**第4-5个月**

#### API聚合器

- 月固定成本：€1,350
- ARPU：€40
- 变动成本/用户：€2（API调用+支付）
- **盈亏平衡点**：1,350 / (40 - 2) = **36个付费用户**

预计达到时间：**第3-4个月**

#### Chrome扩展

- 月固定成本：€220
- ARPU：€4.99
- 变动成本/用户：€0.30
- **盈亏平衡点**：220 / (4.99 - 0.30) = **47个付费用户**

预计达到时间：**第5-6个月**
