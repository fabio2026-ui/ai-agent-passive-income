# 🔧 正确的命令格式

========================================
❌ 还是格式错误
========================================

你执行了：
```bash
pkill -9 Python sleep 3 python3 ~/ai_secure_v2.py &
```

这会把所有内容都当作pkill的参数！

========================================
✅ 正确的格式
========================================

**使用 && 分隔：**

```bash
pkill -9 Python && sleep 3 && python3 ~/ai_secure_v2.py &
```

**或者使用分号 ; ：**

```bash
pkill -9 Python; sleep 3; python3 ~/ai_secure_v2.py &
```

========================================
⚡ 立即执行
========================================

**复制这一行（完全复制）：**

```bash
pkill -9 Python && sleep 3 && python3 ~/ai_secure_v2.py &
```

========================================
✅ 验证
========================================

执行后检查：
```bash
lsof -i :9999
```
