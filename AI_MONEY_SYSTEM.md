## 🤖 建议创建的虚拟机器人团队

### 机器人1：虚拟用户测试员（User Tester Bot）
**功能：**
- 模拟真实用户使用APP
- 记录操作路径、卡顿、崩溃
- 生成用户体验报告
- 截图/录屏留存证据

**测试维度：**
```
1. 首次使用体验（Onboarding）
2. 核心功能测试
3. 边界情况（极端输入）
4. 性能测试（响应时间）
5. 情感反馈（通过日志分析）
```

### 机器人2：视频观众模拟器（Viewer Simulator）
**功能：**
- 模拟不同用户观看视频
- 记录观看时长、点击率
- 预测完播率
- 生成观众画像

**模拟类型：**
```
- 快速滑过型（3秒跳过）
- 好奇观看型（看30秒）
- 深度沉浸型（看完+点赞）
- 批判评论型（看完+吐槽）
```

### 机器人3：多语言客服代表（Multilingual Agent）
**功能：**
- 中文、英文、意大利语客服
- 自动翻译回复
- 处理常见问题
- 转接人工（复杂问题）

### 机器人4：内容审核员（Content Moderator）
**功能：**
- 审核视频/评论是否违规
- 检测敏感词
- 自动标记问题内容
- 生成审核报告
