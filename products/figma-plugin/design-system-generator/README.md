## 功能
一键生成完整的设计系统：
- 🎨 颜色系统 (14个颜色)
- 🔤 字体层级 (10个层级)
- 📏 间距系统 (12个间距)
- ☁️ 阴影样式 (5个阴影)
- 🧩 基础组件 (按钮/输入框/卡片)
