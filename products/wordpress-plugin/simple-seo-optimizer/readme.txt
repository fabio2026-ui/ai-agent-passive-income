== Description ==

Simple SEO Optimizer帮助您轻松优化WordPress网站的SEO设置，无需技术知识。

**主要功能：**

* 一键批量优化所有文章SEO
* 自动生成SEO友好的标题
* 智能提取文章描述
* 自动关键词分析
* 单独文章SEO设置
* SEO统计面板

**免费版功能：**
* 基础SEO优化
* 单篇文章设置
* 批量优化（最多10篇/次）
* 标准关键词提取

**Pro版功能（€29）：**
* 无限批量优化
* 高级关键词分析
* 竞争对手分析
* SEO评分系统
* XML站点地图生成
* 404页面监控
* 重定向管理
* 优先支持

== Installation ==

1. 上传插件到 `/wp-content/plugins/` 目录
2. 在WordPress后台激活插件
3. 访问 "Simple SEO" 菜单开始使用
