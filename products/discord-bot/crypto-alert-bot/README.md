# Discord Crypto Alert Bot
# 小七团队开发

## 功能
- 实时监控加密货币价格
- 价格达到目标时DM提醒
- 免费版: 5个监控/60分钟检查
- Pro版: 100个监控/5分钟检查 (€5/月)

## 部署
```bash
export DISCORD_BOT_TOKEN='your_token'
pip install -r requirements.txt
python bot.py
```

## 命令
- `!price bitcoin` - 查价格
- `!alert bitcoin above 50000` - 设置提醒
- `!alerts` - 查看提醒列表
- `!pricing` - 查看定价
