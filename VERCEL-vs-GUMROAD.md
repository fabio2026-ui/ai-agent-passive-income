# Vercel vs Gumroad - 快速对比

## Vercel（技术部署平台）

### 是什么
- 前端网站托管平台
- 自动部署代码到全球CDN
- 支持Next.js/React/Vue等

### 用在哪些项目
| 项目 | 用途 |
|------|------|
| 营销落地页 | 展示产品、收集邮箱 |
| API文档站 | 展示API使用说明 |
| 仪表盘 | 收入/数据可视化 |
| 个人主页 | 艾琳IP展示网站 |

### 为什么需要
- 免费托管
- 自动SSL
- 全球加速
- 自动部署

---

## Gumroad（数字产品销售平台）

### 是什么
- 卖虚拟产品的电商平台
- 用户付费后自动下载
- 支持PDF/代码/模板等

### 用在哪些项目
| 项目 | 产品 | 价格 |
|------|------|------|
| Notion模板 | 3个模板包 | €15-29 |
| UI Kit | 设计系统源文件 | €29-49 |
| Chrome扩展 | 代码包+安装指南 | €3-9 |
| API服务 | 开发者订阅 | €5-29/月 |
| 电子书 | 副业指南 | €9-19 |

### 为什么需要
- 收钱工具
- 自动交付
- 邮件营销
- affiliate分销

---

## 一句话总结
- **Vercel** = 放网站的地方（技术）
- **Gumroad** = 卖东西收钱的地方（商业）
