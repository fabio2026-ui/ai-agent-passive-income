# 📺 TeamViewer 远程控制方案

========================================
✅ 为什么选择TeamViewer
========================================

优点：
- ✅ 成熟稳定，不会断
- ✅ 你可以主动发起连接
- ✅ 我可以完全控制你的Mac
- ✅ 包括浏览器、文件、所有操作
- ✅ 不需要复杂配置

========================================
🔧 安装步骤 (5分钟)
========================================

## 步骤1: 下载TeamViewer

**浏览器访问：**
```
https://www.teamviewer.com/en/download/mac/
```

或者直接下载：
```
https://download.teamviewer.com/download/TeamViewer.dmg
```

## 步骤2: 安装

1. 打开下载的 .dmg 文件
2. 双击 TeamViewer 图标安装
3. 按照提示完成安装

## 步骤3: 启动并获取ID

1. 打开 TeamViewer 应用
2. 选择 "远程控制" 模式
3. 会显示：
   - **你的ID** (9位数字，如：123 456 789)
   - **密码** (4位数字)

========================================
🔗 建立连接
========================================

## 方式1: 你告诉我ID和密码

```
TeamViewer ID: xxx xxx xxx
Password: xxxx
```

我输入ID和密码，连接你的Mac。

## 方式2: 你主动邀请我

1. 点击 "会议/演示" 
2. 选择 "邀请参与者"
3. 复制链接发给我

========================================
⚠️ 安全提醒
========================================

- 只在需要时开启TeamViewer
- 用完后可以关闭或改密码
- 我可以控制你的鼠标键盘
- 你可以随时断开连接

========================================

========================================

**回复：**

**"下载中"** → 去下载安装TeamViewer

**"已安装"** → 告诉我你的ID和密码

**"不会装"** → 你先手动完成Fiverr，TeamViewer以后再说
