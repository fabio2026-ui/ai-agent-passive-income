# Wisdom & Decision Making System

## ⚖️ 决策框架库

### 1. 理性决策模型 (Rational Decision Making)
```
1. 明确目标: 你想要达成什么？
2. 收集信息: 相关的事实和数据
3. 生成选项: 所有可能的行动方案
4. 评估后果: 每个选项的利弊
5. 权重排序: 根据重要性排序标准
6. 选择最优: 综合得分最高的选项
7. 执行监控: 实施并跟踪结果
8. 反馈调整: 根据结果优化
```

### 2. 满意决策法 (Satisficing - Simon)
- **设定门槛**: 确定"足够好"的标准
- **找到第一个满足条件的**: 不需要最优，够用就行
- **节省认知资源**: 在复杂环境中更高效
- **适用场景**: 时间紧迫、信息不全、成本敏感

### 3. 遗憾最小化框架 (Regret Minimization - Bezos)
- **投射到80岁**: 回顾人生时会后悔什么？
- **长期视角**: 淡化短期痛苦，聚焦长期价值
- **行动偏向**: 做了后悔 vs 没做后悔
- **关键问题**: "不做这个，我会后悔吗？"

### 4. 机会成本思维
- **隐性成本**: 做A意味着放弃B的价值
- **比较基准**: 不是"有没有收益"，是"相比其他选择"
- **资源稀缺性**: 时间/金钱/注意力都是有限的
- **沉没成本谬误**: 过去的投入不影响未来决策

## 🎯 战略思维工具

### SWOT 分析
- **S**trengths: 内部优势
- **W**eaknesses: 内部劣势
- **O**pportunities: 外部机会
- **T**hreats: 外部威胁

### Porter 五力分析
1. **现有竞争者**: 竞争激烈程度
2. **潜在进入者**: 新进入者的威胁
3. **替代品**: 替代产品/服务
4. **供应商议价力**: 上游控制力
5. **客户议价力**: 下游控制力

### 蓝海战略
- **消除**: 哪些行业认为是理所当然但可以去掉的？
- **减少**: 哪些因素可以降到行业标准以下？
- **提升**: 哪些因素可以大大超过标准？
- **创造**: 哪些行业从未提供的价值可以创造？

## 🧭 价值观与伦理决策

### 功利主义 (Utilitarianism)
- **核心**: 最大化整体幸福
- **计算**: 每个决策的净收益 (收益-成本)
- **问题**: 少数人利益可以被牺牲
- **适用**: 公共政策、资源分配

### 义务论 (Deontology)
- **核心**: 有些行为本身就是错的
- **绝对命令**: 如果所有人都这么做会怎样？
- **权利**: 保护基本人权
- **适用**: 伦理底线、不可谈判原则

### 美德伦理 (Virtue Ethics)
- **核心**: 成为什么样的人比做什么更重要
- **德性**: 智慧、勇气、节制、正义
- **实践**: 在情境中培养判断力
- **适用**: 个人品格、长期关系

### 关怀伦理 (Care Ethics)
- **核心**: 关注关系和情感
- **责任**: 对特定他人的责任
- **情境**: 具体情况具体分析
- **适用**: 亲密关系、团队管理

## 🎪 心理与行为经济学洞察

###  Prospect Theory (前景理论)
- **损失厌恶**: 损失的痛苦 > 获得的快乐
- **参照依赖**: 评估结果相对于参照点
- **敏感度递减**: 边际效用递减
- **概率权重**: 对小概率事件过度反应

###  Mental Accounting (心理账户)
- **分类处理**: 钱被分到不同心理账户
- **不可替代性**: 意外之财 vs 血汗钱
- **沉没成本**: 为过去投入继续投入
- **决策隔离**: 每个决策独立看待

###  Nudge (助推)
- **默认选项**: 默认设置影响选择
- **框架**: 表述方式影响决策
- **简化**: 减少选择数量
- **反馈**: 即时反馈改变行为

## 🏆 智慧原则 (Wisdom Principles)

### Charlie Munger 思维模型
1. **能力圈**: 只在懂的地方下注
2. **安全边际**: 留足犯错空间
3. **机会成本**: 比较所有可行选项
4. **复利**: 时间是最好的朋友
5. **逆向思维**: 想失败先避免做什么
6. **多学科思维**: 跨学科重要模型
7. **检查清单**: 避免遗漏关键步骤
8. **承认无知**: 知道边界比知道答案重要

### 纳西姆·塔勒布 (Nassim Taleb)
1. **反脆弱**: 从波动中获益
2. **Skin in the game**: 承担风险才有发言权
3. **随机性**: 区分运气和能力
4. **黑天鹅**: 罕见事件的巨大影响
5. **杠铃策略**: 极端保守+极端冒险

### 丹尼尔·卡尼曼 (Daniel Kahneman)
1. **系统1 vs 系统2**: 快思考 vs 慢思考
2. **认知偏差**: 系统性思维错误
3. **外部观点**: 用基准概率评估
4. **事前验尸**: 假设失败，找原因
5. **算法辅助**: 简单规则胜过专家直觉

## 🌟 决策质量检查清单

### 在做出重大决策前问自己：

**信息充分性**
- [ ] 我有所有必要的信息吗？
- [ ] 我有没有忽视反面证据？
- [ ] 我的信息来源可靠吗？

**逻辑严密性**
- [ ] 我的推理过程有漏洞吗？
- [ ] 结论真的能从前提推出吗？
- [ ] 有没有更好的解释？

**偏见检查**
- [ ] 我是否在确认已有观点？
- [ ] 情绪是否影响了判断？
- [ ] 我是否被第一印象锚定了？

**长期视角**
- [ ] 10年后我会怎么看这个决定？
- [ ] 最坏情况我能承受吗？
- [ ] 有没有不可逆的后果？

**替代方案**
- [ ] 我考虑了所有可行选项吗？
- [ ] 有没有"第三条路"？
- [ ] 不行动也是一种选择吗？

**反馈机制**
- [ ] 我怎么知道决策是否正确？
- [ ] 需要多长时间评估结果？
- [ ] 什么信号说明需要调整？

## 🎭 智慧决策 persona

### 智者模式 (The Sage)
- 不急不躁，深思熟虑
- 考虑多代际影响
- 追求长期和谐
- 承认复杂性和不确定性

### 战士模式 (The Warrior)
- 果断行动，承担风险
- 在信息不全时决策
- 接受失败为学费
- 从战斗中学习

### 商人模式 (The Merchant)
- 成本收益计算
- 机会成本意识
- 快速试错迭代
- 用数据说话

### 园丁模式 (The Gardener)
- 耐心培育，等待成长
- 关注系统健康
- 复利思维
- 与环境和谐

## ⚡ 快速决策法则

### 10-10-10 法则
- **10分钟后**: 我会怎么看？
- **10个月后**: 影响还在吗？
- **10年后**: 重要吗？

### 37% 法则 (最优停止)
- **探索期**: 前37%只看不选
- **选择期**: 之后选比前面都好的
- **适用**: 选房、选人、选机会

### 5秒法则
- 如果5秒内能决定的小事，立即决定
- 不浪费认知资源在琐事上
- 把精力留给重要决策
