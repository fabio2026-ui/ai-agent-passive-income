# 小七精品产品用户研究与推广策略
# 先研究用户场景和价值主张，再执行推广

## 🎯 研究原则
1. **用户第一**: 先理解用户是谁，他们在什么场景下使用
2. **痛点驱动**: 用户为什么选择我们？解决什么痛点？
3. **差异化清晰**: 与竞品相比，我们的独特价值是什么？
4. **场景具体**: 不是"提高效率"，而是"周五下午3点审核报表时..."

---

## 📊 精品产品用户研究

### 产品1: Amazon Calculator API

#### 用户画像
| 维度 | 描述 |
|------|------|
| **用户类型** | Amazon FBA卖家 |
| **用户规模** | 中小型卖家 (月销售额$10K-$500K) |
| **地理位置** | 欧盟国家 (德国、法国、意大利、西班牙等) |
| **痛点** | 算不清VAT和各项费用，利润被吃掉 |
| **年龄** | 25-45岁 |
| **技术能力** | 中等，会用Excel但不懂编程 |

#### 使用场景
**场景1: 选品决策**
> 李想在Amazon上卖电子产品，看到一款耳机进货价€20，售价€50。
> 但他不知道扣掉VAT、FBA费用、佣金后还剩多少利润。
> 用我们的API一算：净利润€8.5，利润率17%，决定不做这个产品。

**场景2: 月度财务核算**
> 月底对账时，王老板发现销售额€50K，但利润只有€3K。
> 用我们的API反向核算，发现VAT计算有误，及时调整了定价策略。

**场景3: 多国家运营**
> 张 seller 同时在德国、法国、意大利销售。
> 每个国家VAT率不同，费用结构不同，人工计算容易出错。
> 用我们的API一键计算三个国家的利润。

#### 用户为什么选择我们？

| 痛点 | 现有解决方案 | 我们的差异化价值 |
|------|-------------|------------------|
| VAT计算复杂 | 手动Excel计算 | ✅ 自动计算，准确率100% |
| 费用不透明 | Amazon后台数据分散 | ✅ 一站式清晰展示 |
| 多国家难管理 | 分别查询各国政策 | ✅ 一个API覆盖欧盟多国 |
| 实时性要求 | 下载报表后再计算 | ✅ 实时API查询 |

**核心卖点**: 
> "3秒钟算出你的真实利润，不再被VAT和隐藏费用吃掉利润"

### 产品2: Shopify Calculator API

#### 用户画像
| 维度 | 描述 |
|------|------|
| **用户类型** | Shopify独立站卖家 |
| **用户规模** | 中小型电商 (月销售额$5K-$200K) |
| **地理位置** | 全球，特别是美国、英国、欧盟 |
| **痛点** | 不知道扣掉支付手续费、物流费后真实利润 |
| **特点** | 更关注品牌，但财务意识较弱 |

#### 使用场景
**场景1: 定价策略制定**
> 新品牌上线，产品成本$15，想卖$45。
> 用API一算：扣掉Shopify费用、PayPal手续费、物流费，净利润$18，利润率40%。
> 确认这个定价可行。

**场景2: 促销活动评估**
> 黑五想做8折促销，但不知道还能不能赚钱。
> 用API模拟计算：8折后净利润$8，利润率降至18%，决定只做9折。

**场景3: 跨境扩张决策**
> 想开拓欧盟市场，但不知道税务成本。
> 用API计算：VAT 20%，加上物流成本，利润压缩到12%，决定暂缓。

#### 用户为什么选择我们？

| 痛点 | 现有解决方案 | 我们的差异化价值 |
|------|-------------|------------------|
| 费用计算复杂 | Shopify后台数据分散 | ✅ 一键计算所有费用 |
| 跨境税务不清楚 | 咨询会计(贵) | ✅ 自动计算各国VAT |
| 促销决策难 | 凭感觉定价 | ✅ 数据支撑决策 |

**核心卖点**: 
> "Shopify卖家的财务计算器，让每一笔订单的利润都清晰可见"

### 产品3: Breathing AI (App)

#### 用户画像
| 维度 | 描述 |
|------|------|
| **用户类型** | 职场白领、焦虑人群、睡眠困难者 |
| **年龄** | 22-40岁 |
| **地理位置** | 全球，主要是发达国家 |
| **痛点** | 压力大、焦虑、失眠、无法专注 |
| **行为特点** | 用过Headspace/Calm但觉得太复杂或太贵 |

#### 使用场景
**场景1: 工作间隙放松**
> 下午3点，连续开了3个会，大脑转不动了。
> 打开Breathing AI，选择"快速放松"模式，2分钟呼吸练习。
> 重新集中注意力，继续工作。

**场景2: 睡前助眠**
> 晚上11点，躺床上刷手机越刷越清醒。
> 打开Breathing AI，选择"助眠呼吸"，跟着节奏呼吸5分钟。
> 自然入睡。

**场景3: 焦虑发作时**
> 突然感到心慌、焦虑。
> 打开Breathing AI，选择"缓解焦虑"模式。
> 4-7-8呼吸法，3分钟后平静下来。

#### 用户为什么选择我们？

| 痛点 | Headspace/Calm | 我们的差异化价值 |
|------|----------------|------------------|
| 太贵 | $69.99/年 | ✅ 免费或低价 |
| 太复杂 | 需要学习课程 | ✅ 打开即用，无需学习 |
| 太重 | App体积大 | ✅ 轻量级Web App |
| 功能冗余 | 冥想、瑜伽、睡眠都有 | ✅ 专注呼吸，极简体验 |

**核心卖点**: 
> "不需要学习，不需要订阅，打开就能用的呼吸助手"

### 产品4: SEO Analyzer Pro (Chrome扩展)

#### 用户画像
| 维度 | 描述 |
|------|------|
| **用户类型** | 网站运营、内容创作者、SEO新手 |
| **用户规模** | 个人站长到中小企业 |
| **地理位置** | 全球 |
| **痛点** | 不知道页面SEO哪里做错了 |
| **技术能力** | 懂一点SEO但不够专业 |

#### 使用场景
**场景1: 发布文章前检查**
> 写了一篇博客文章，准备发布。
> 用SEO Analyzer检查：发现缺少meta description，H1标题太长，图片没有alt标签。
> 修正后发布。

**场景2: 竞品分析**
> 想看看竞争对手的页面为什么排名高。
> 用SEO Analyzer分析：发现对方的关键词密度、内链结构优势。
> 学习优化自己的页面。

**场景3: 网站健康检查**
> 接手一个新网站，想快速了解SEO状况。
> 用SEO Analyzer扫描首页：发现5个关键问题，制定优化计划。

#### 用户为什么选择我们？

| 痛点 | 现有工具(如Ahrefs) | 我们的差异化价值 |
|------|-------------------|------------------|
| 太贵 | $99+/月 | ✅ 免费或一次性付费 |
| 太重 | 功能太多用不上 | ✅ 轻量级，聚焦基础SEO |
| 太慢 | 需要登录后台 | ✅ 浏览器一键分析 |
| 太专业 | 需要SEO知识 | ✅ 新手友好，直接指出问题 |

**核心卖点**: 
> "一键检查页面SEO问题，新手也能做出专业优化"

## ⚠️ 风险提示

1. **市场教育成本**: 用户可能不知道需要这个工具
2. **竞品反应**: 大厂可能快速复制功能
3. **获客成本**: 小众市场获客可能不划算
4. **变现验证**: 免费用户转付费的比例未知
