## 统一设计规范

### 字体建议
- **标题**: Inter, Montserrat, 或思源黑体 Bold
- **正文**: Inter, Open Sans, 或思源黑体 Regular
- **英文**: 与中文搭配协调的无衬线字体

### 价格标签样式（统一）
```
形状：圆形
尺寸：120-150px直径
位置：右下角，距边缘40-60px
背景：产品主色
文字：€XX，白色粗体，36-48px
阴影：添加轻微阴影提升层次感
```

### Notion Logo使用
```
位置：左下角，距边缘40px
大小：60-80px宽
样式：彩色原版logo
透明度：100%
```

### 安全区域
```
Gumroad会在图片上叠加文字和按钮，
确保重要内容（如主标题）位于：
- 距离顶部至少100px
- 距离底部至少150px
- 距离左右边缘至少80px
```

## 快速制作流程（Canva）

1. 访问 canva.com
2. 点击"创建设计" → "自定义尺寸" → 1280x720
3. 选择背景色或渐变
4. 添加标题文字（使用模板中的文字）
5. 添加装饰元素（形状、线条、icon）
6. 上传Notion logo和价格标签
7. 调整位置和大小
8. 下载为PNG格式
9. 检查文件大小（建议<500KB）

## 设计检查清单

发布前确认：
- [ ] 图片尺寸正确（1280x720）
- [ ] 文字清晰可读（在小缩略图也能看清）
- [ ] 价格标签醒目
- [ ] 整体风格与产品调性匹配
- [ ] 背景不会干扰文字阅读
- [ ] 颜色对比度足够
- [ ] 文件格式为PNG或JPG
- [ ] 文件大小适中（<1MB）
