## 一、黄金比例 φ = 1.618033988749895...

### 数学本质
黄金比例是方程 x² = x + 1 的正解：
```
φ = (1 + √5) / 2 ≈ 1.618
```

**独特性质**：
- φ = 1 + 1/φ
- φ² = φ + 1
- 1/φ = φ - 1 ≈ 0.618

### 自然界中的黄金比例
| 现象 | 描述 |
|------|------|
| 鹦鹉螺壳 | 对数螺旋，每转增长φ倍 |
| 向日葵种子 | 斐波那契螺旋排列 |
| 人脸比例 | 五官符合φ比例更"美" |
| 银河系旋臂 | 对数螺旋结构 |

### 设计应用
```
版面分割:  62% : 38%
字体层级:  16px → 26px → 42px → 68px
间距系统:  8 → 13 → 21 → 34 → 55
```

## 二、瑞士网格系统 (Swiss Grid)

### 历史背景
- **起源**: 1950s 瑞士国际主义风格 (International Typographic Style)
- **代表**: Josef Müller-Brockmann, Armin Hofmann
- **核心**: 客观、秩序、数学精度

### 12列网格的数学
```
总宽度: 1440px
边距:   144px (10%)
内容区: 1152px
列宽:   80px  (1152 / 12 - 间距)
间距:   32px  (黄金比例: 80 × 0.4)
```

### 网格的数学美
- **可整除性**: 12可被2,3,4,6整除，灵活布局
- **韵律感**: 固定间距创造视觉节奏
- **约束即自由**: 限制创造一致性

## 三、色彩数学

### HSL色彩空间
比RGB更符合人类感知：
- **H**ue (色相): 0-360° 色轮角度
- **S**aturation (饱和度): 0-100% 纯度
- **L**ightness (亮度): 0-100% 明暗

### 色彩和谐公式

**互补色 (Complementary)**
```
和谐色 = (H + 180) % 360
对比度最高，视觉冲击强
```

**三色 (Triadic)**
```
色1 = H
色2 = (H + 120) % 360
色3 = (H + 240) % 360
平衡且丰富
```

**分裂互补 (Split)**
```
主色 = H
辅色1 = (H + 150) % 360
辅色2 = (H + 210) % 360
高对比但和谐
```

**类比色 (Analogous)**
```
色1 = (H - 30) % 360
主色 = H
色2 = (H + 30) % 360
和谐自然
```

### WCAG对比度数学
```
对比度 = (L1 + 0.05) / (L2 + 0.05)

AA级: 4.5:1 (正文), 3:1 (大标题)
AAA级: 7:1 (正文), 4.5:1 (大标题)
```

## 四、排版几何学

### 基线网格 (Baseline Grid)
**概念**: 所有文字对齐到隐形的水平线网格

**计算**:
```
基础字号: 16px
行高:     1.5 = 24px
段落间距: 24px (保持 rhythm)

H1: 48px / 48px (2倍行高)
H2: 36px / 48px (1.5倍行高)
H3: 24px / 24px (1倍行高)
Body: 16px / 24px
```

### 垂直节奏 (Vertical Rhythm)
**原理**: 所有间距是基线单位的整数倍

**数学**:
```
间距 = n × 基线单位 (n ∈ ℕ)
常用: 8, 16, 24, 32, 48, 64, 96...
```

**为什么8pt?**
- 可被2,4整除，灵活分割
- 主流屏幕分辨率可被8整除
- 视觉上"刚好"的增量

### 黄金比例排版
```
基础: 16px
1级:  16 × 1.618 = 25.9px ≈ 26px
2级:  26 × 1.618 = 42.1px ≈ 42px
3级:  42 × 1.618 = 68.0px ≈ 68px
```

## 五、模块化比例 (Modular Scale)

### 音乐比例
设计中的比例借鉴音乐理论：

| 比例名 | 值 | 音乐对应 | 感觉 |
|--------|-----|----------|------|
| Minor Second | 1.067 | 半音 | 微妙 |
| Major Second | 1.125 | 全音 | 温和 |
| Minor Third | 1.200 | 小三度 | 和谐 |
| Major Third | 1.250 | 大三度 | 明亮 |
| Perfect Fourth | 1.333 | 纯四度 | 平衡 |
| Perfect Fifth | 1.500 | 纯五度 | 开放 |
| Golden | 1.618 | - | 自然 |
| Octave | 2.000 | 八度 | 强烈 |

### 应用
```python
# 基于黄金比例的字体系统
base = 16
scale = [base * (1.618 ** i) for i in range(-2, 6)]
# → [6.1, 9.9, 16, 25.9, 41.9, 67.8, 109.7, 177.4]
```

## 六、格式塔心理学 (Gestalt Principles)

### 1. 接近律 (Proximity)
**原理**: 距离近的物体被视为一组

**数学**: 组内间距 < 组间间距 × 0.5

### 2. 相似律 (Similarity)
**原理**: 形状/颜色/大小相似的物体被视为一组

### 3. 连续律 (Continuity)
**原理**: 眼睛沿平滑路径移动

**应用**: 对齐比居中更重要

### 4. 闭合律 (Closure)
**原理**: 大脑自动补全不完整形状

**应用**: 简洁图标设计

### 5. 图形-背景 (Figure-Ground)
**原理**: 视觉自动分离前景/背景

**应用**: 卡片式设计、阴影层次

## 八、视觉层次数学

## 十、设计工具箱

### 黄金比例计算器
```python
phi = 1.618033988749895

def golden_section(n):
    return n / phi, n - (n / phi)
```

### 色彩对比度检查
```python
def contrast(rgb1, rgb2):
    L1 = luminance(rgb1)
    L2 = luminance(rgb2)
    return (max(L1,L2) + 0.05) / (min(L1,L2) + 0.05)
```
