# 🔧 配置OpenClaw节点 - 系统级控制

========================================
🎯 目标
========================================

建立稳定的系统级控制，替代不稳定的Python服务器。

========================================
🔧 配置步骤
========================================

## 步骤1: 启动节点（在你的Mac执行）

```bash
openclaw node start
```

应该会显示：
```
Starting node...
Node started on http://127.0.0.1:xxxxx
```

## 步骤2: 获取配对码

```bash
openclaw node pair
```

会显示：
```
Pairing code: ABC123XYZ
```

**复制这个配对码发给我！**

## 步骤3: 我在服务器端批准

我会执行：
```bash
openclaw node approve ABC123XYZ
```

## 步骤4: 验证连接

我会测试：
```bash
openclaw node run fabiomacbook-air -- ls ~/
```

如果成功，就可以稳定控制你的Mac！

========================================
📋 现在执行
========================================

**在你的Mac终端执行：**

```bash
openclaw node start
```

**然后把输出发给我！**
