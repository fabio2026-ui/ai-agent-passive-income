# 🔧 修正命令

========================================
❌ 命令拼写错误
========================================

你执行了：
```bash
tailscale funnel statu
```

缺少了一个字母 `s`！

========================================
✅ 正确的命令
========================================

```bash
tailscale funnel status
```

注意：**status** 不是 **statu**

========================================
⚡ 立即执行
========================================

**复制粘贴：**

```bash
tailscale funnel status
```
