# 微信小程序申请快速指南

## 申请地址
https://mp.weixin.qq.com/

## 准备材料
- 邮箱 (未注册过微信的)
- 手机号
- 身份证正反面照片
- 个人/企业信息

## 申请步骤
1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册"
3. 选择"小程序"
4. 填写邮箱、密码
5. 邮箱激活
6. 信息登记
7. 等待审核 (通常1-3个工作日)

## 小程序信息
- 名称: ContentAI创作助手
- 简称: ContentAI
- 类目: 工具-效率工具
- 介绍: AI内容创作助手，10秒生成小红书/抖音爆款文案

## 快速开始
```bash
# 安装微信小程序开发者工具
# https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
```
