Hi HN,

I built ContentAI, an AI content generator that uses the Bring Your Own Key (BYOK) model to achieve zero operating costs.

**Why BYOK?**
Most AI writing tools charge $20-50/month and impose usage limits. But what if you only need AI occasionally? I built ContentAI for people like me who want to use AI tools without committing to another monthly subscription.

**How it works:**
1. You provide your own Moonshot AI API key
2. Everything runs in your browser - no backend server
3. No data is stored anywhere
4. Use it as much as your API key allows

**Technical details:**
- Pure HTML/CSS/JS (no frameworks)
- Single HTML file, <100KB total
- IPFS deployed for decentralized hosting
- No build step, no dependencies to manage
- Response streaming for real-time generation
