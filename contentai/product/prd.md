## 3. 功能规格

### 3.1 核心功能

#### 功能1: 智能文案生成
**用户故事**: 作为小红书博主，我想输入产品名称，让AI帮我写一篇种草文案

**输入**: 
- 选择平台（小红书/抖音/公众号）
- 输入主题/产品名
- 输入关键词（可选）
- 选择风格（专业/轻松/幽默）
- 选择长度（短/中/长）

**输出**:
- 标题（带emoji）
- 正文（分段，有节奏感）
- 话题标签

**示例**:
```
输入:
- 平台: 小红书
- 主题: 一款国产护肤精华
- 关键词: 平价、美白、学生党
- 风格: 轻松
- 长度: 中等

输出:
【标题】学生党必入！💰69的美白精华也太香了吧！

【正文】
姐妹们！我发现了一个宝藏精华！✨

作为一个黄黑皮，美白真的是终身课题...
...

【标签】
#美白精华 #学生党护肤 #平价好物
```

#### 功能2: 多平台适配
**小红书风格**:
- 标题党 + emoji
- 分段清晰，多用短句
- 干货满满，种草感强
- 话题标签10-15个

**抖音风格**:
- 开头黄金3秒
- 口语化，有节奏
- 情绪起伏
- 引导互动

**公众号风格**:
- 深度长文
- 逻辑清晰
- 金句频出
- 排版美观

#### 功能3: 文案优化
- 续写: 在现有基础上继续写
- 改写: 换一种表达方式
- 扩写: 简短变详细
- 缩写: 详细变简短
- 换风格: 正式变轻松等

#### 功能4: 历史记录管理
- 保存所有生成记录
- 支持收藏
- 支持分类（按平台/标签）
- 支持搜索
- 支持删除/恢复

#### 功能5: 模板市场
预设热门模板:
- 小红书: 种草文、探店文、教程文
- 抖音: 带货脚本、剧情脚本、知识分享
- 公众号: 深度长文、热点评论

## 4. 用户流程

### 4.1 新用户流程
```
进入小程序 → 选择平台 → 输入主题 → 点击生成 
→ 查看结果 → （可选：修改/重新生成）→ 复制使用
→ 引导注册 → 提示剩余次数 → 引导付费
```

### 4.2 付费用户流程
```
登录 → 首页 → 选择平台 → 输入主题 → 生成 
→ 满意 → 保存/复制
→ 不满意 → 调整参数重新生成
```

## 8. 竞品分析

### 8.1 直接竞品
| 竞品 | 优点 | 缺点 | 我们的机会 |
|------|------|------|-----------|
| 小红书创作助手 | 官方，数据好 | 功能简单 | 更智能的生成 |
| 抖音创作服务 | 官方支持 | 只支持抖音 | 多平台支持 |
| Writesonic中文版 | 功能多 | 中文理解差 | 中文优化 |

### 8.2 差异化优势
1. 专为中文社交媒体优化
2. 理解各平台爆款逻辑
3. 支持多平台一键适配
4. 价格更亲民
