## 🎯 标题组合公式

### 高点击率标题结构

```
【数字】+【痛点/结果】+【解决方案】
例：3个月涨粉5万，我只做了这1件事

【悬念】+【揭秘】+【价值】
例：小红书博主都在用的秘密武器，今天终于公开

【对比】+【反差】+【好奇】
例：花了2小时写的文案，不如AI 3秒生成的效果好

【身份】+【经历】+【转变】
例：从被裁员到月入5万，小红书改变了我的命运
```

### 标题优化检查清单

- [ ] 是否包含数字？
- [ ] 是否有情绪词汇？
- [ ] 是否制造了悬念/好奇？
- [ ] 是否承诺了具体价值？
- [ ] 是否针对特定人群？
- [ ] 是否在20字以内？
- [ ] 是否有emoji增加视觉吸引力？
- [ ] 是否避免了标题党过度夸张？
