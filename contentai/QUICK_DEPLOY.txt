# ContentAI - 2分钟快速部署指南

## 🚀 最快部署方式 (Netlify Drop)

### 步骤 (2分钟)

1. **打开Netlify Drop**
   ```
   https://app.netlify.com/drop
   ```

2. **上传文件**
   ```
   拖拽文件夹: /root/.openclaw/workspace/contentai/final-deploy
   到网页中
   ```

3. **完成！**
   - 自动获得域名: xxx.netlify.app
   - 立即访问使用
