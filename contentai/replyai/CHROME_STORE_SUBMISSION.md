### 详细描述

**ReplyAI - 您的智能邮件回复助手**

告别邮件回复烦恼！ReplyAI是一款强大的Chrome扩展，利用AI技术帮您一键生成专业、得体的邮件回复。

**✨ 核心功能**

🎯 **一键智能回复**
- 自动读取Gmail/Outlook邮件内容
- 基于上下文生成专业回复
- 支持一键插入或复制

🎨 **多种回复风格**

- 😊 友好亲切 - 轻松自然的语气
- ⚡ 简洁明了 - 直击要点
- ✨ 自定义 - 描述您的专属风格

📧 **完美支持主流邮箱**
- Gmail (mail.google.com)
- Outlook (outlook.live.com)
- Office 365 (outlook.office.com)

🚀 **零成本运营**
- 用户自带OpenAI API Key
- 无订阅费用
- 按实际使用量付费

🛡️ **隐私保护**
- API Key仅存储在本地
- 邮件内容不上传第三方服务器
- 直接调用OpenAI API

📚 **快捷模板**
内置5种常用邮件模板：感谢、确认、道歉、跟进、拒绝，一键生成专业模板。

**💰 费用说明**

ReplyAI本身是免费工具。使用OpenAI API会产生费用：
- GPT-4o Mini: ~$0.00015/1K tokens
- 一封普通邮件: ~$0.001-0.003

**🔧 使用方法**

1. 安装扩展后，点击图标输入OpenAI API Key
2. 打开Gmail/Outlook的任意邮件
3. 点击"读取当前邮件"
4. 选择回复风格，点击"生成智能回复"
5. 一键插入或复制到剪贴板

## 🔒 隐私政策

**ReplyAI 隐私政策**

生效日期: 2024年

**数据收集**

ReplyAI尊重用户隐私，采取最小化数据收集原则：

1. **本地存储数据**
   - OpenAI API Key（用户主动输入）
   - 用户偏好设置（回复风格等）
   - 这些数据仅存储在您的浏览器本地，不会上传到任何服务器

2. **邮件内容**
   - 邮件内容仅在本地浏览器处理
   - 生成回复时会发送到OpenAI API（用户自己的API Key）
   - 我们不会存储、收集或分析您的邮件内容

3. **不收集的数据**
   - 不收集用户身份信息
   - 不收集浏览历史
   - 不收集任何个人敏感信息

**数据使用**

- 邮件内容仅用于生成回复，通过OpenAI API处理
- 遵循OpenAI的隐私政策和服务条款
- 不会将数据用于广告或营销目的

**数据安全**

- 所有数据存储在浏览器本地storage中
- 使用Chrome浏览器提供的安全机制
- 可随时通过扩展设置清除所有数据

**第三方服务**

本扩展使用以下第三方服务：
- OpenAI API (api.openai.com) - 用于生成邮件回复

**联系我们**

如有隐私相关问题，请联系: support@replyai.app

**变更**

我们可能会更新本隐私政策，更新后会通过扩展内通知告知用户。

## 📄 单一用途说明

ReplyAI的唯一目的是帮助用户生成邮件回复。扩展通过读取当前Gmail/Outlook页面的邮件内容，调用用户自己的OpenAI API Key生成智能回复。

**需要的权限说明：**

1. **storage** - 用于本地存储用户的API Key和设置
2. **activeTab** - 用于读取当前邮件页面的内容
3. **scripting** - 用于在邮件页面执行内容脚本
4. **host_permissions** - 仅限Gmail和Outlook域名，用于邮件内容提取
