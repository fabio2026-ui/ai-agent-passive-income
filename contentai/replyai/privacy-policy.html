🔒 ReplyAI 隐私政策

生效日期: 2024年4月

核心原则: ReplyAI 采取最小化数据收集原则。您的数据主要存储在本地浏览器中，我们不会收集或存储您的邮件内容。

📊 我们收集什么数据

1. 本地存储数据 (仅浏览器本地)

  • OpenAI API Key - 您主动输入的API密钥，用于调用OpenAI服务
  • 用户偏好设置 - 回复风格、默认设置等
  • 存储位置: 浏览器本地 storage，不会上传到我们的服务器

2. 处理过程中的数据

  • 邮件内容 - 仅在您点击"生成回复"时临时处理
  • 处理方式: 通过您的API Key直接发送到OpenAI API
  • 我们不存储: 邮件内容不会存储在我们的服务器上

🚫 我们不收集的数据

  • ❌ 用户身份信息 (姓名、邮箱、电话等)
  • ❌ 浏览历史记录
  • ❌ 个人敏感信息
  • ❌ 设备信息
  • ❌ 位置信息

🔧 数据如何使用

OpenAI API 调用

  • 邮件内容仅用于生成AI回复
  • 通过您自己的API Key直接调用OpenAI服务
  • 遵循 OpenAI隐私政策

数据留存

  • API Key和设置: 保留在本地，直到您主动清除
  • 邮件内容: 处理后立即丢弃，不做任何存储

🛡️ 数据安全

  • 所有本地数据使用Chrome浏览器提供的安全存储机制
  • API传输使用HTTPS加密
  • 可随时通过扩展设置一键清除所有数据

👥 第三方服务

本扩展使用以下第三方服务:

  • OpenAI API (api.openai.com) - 用于生成邮件回复

🔐 您的权利

  • 查看数据: 所有存储的数据都在扩展设置页面可见
  • 删除数据: 可随时一键清除所有本地数据
  • 撤销授权: 可随时卸载扩展，所有本地数据将被删除

📞 联系我们

如有隐私相关问题，请联系:

  • 邮箱: support@replyai.app

📝 政策变更

我们可能会不时更新本隐私政策。重大变更时，我们会通过扩展内通知告知用户。

总结: ReplyAI 是一个隐私优先的工具。您的邮件内容仅用于生成回复，不会被存储或用于其他目的。所有敏感数据都保存在您的本地浏览器中。
