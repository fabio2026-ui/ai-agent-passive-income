# ReplyAI 截图 & 宣传图制作指南

## 📸 截图要求

### 技术规格
- **尺寸**: 1280x800 (推荐) 或 640x400
- **格式**: PNG 或 JPEG
- **数量**: 最少1张，最多5张
- **命名**: screenshot-1.png, screenshot-2.png, ...

### 需要制作的截图

#### 截图1: 主界面展示
**文件名**: `screenshot-main.png`
**尺寸**: 1280x800

**内容布局:**
```
[浏览器窗口 - Gmail页面作为背景]
  ↳ [ReplyAI popup 居中显示]
      - 显示标题 "ReplyAI - 智能客服邮件助手"
      - API Key区域 (显示部分打码的key: sk-...xxxx)
      - 绿色的状态提示 "✅ API Key 已加载"
      - "📥 读取当前邮件" 按钮
```

**制作步骤:**
1. 打开Chrome，进入Gmail
2. 点击ReplyAI扩展图标
3. 输入一个测试API Key
4. 截图整个popup界面
5. 使用Figma/Photoshop调整为1280x800
6. 添加说明文字: "简洁优雅的界面设计"

## 🛠️ 推荐工具

### 免费工具
1. **Figma** (推荐)
   - 在线设计工具
   - 有Chrome Store截图模板
   - 协作方便

2. **Canva**
   - 模板丰富
   - 简单易用
   - 有预设尺寸

3. **GIMP**
   - 免费开源
   - 功能强大
   - 适合精细调整

### 截图工具
- **Chrome DevTools**: F12 → Ctrl+Shift+P → "Capture screenshot"
- **Awesome Screenshot**: Chrome扩展
- **ShareX**: 免费开源 (Windows)

## ✅ 质量检查清单

### 截图检查
- [ ] 图片清晰，无模糊

- [ ] 界面文字可阅读

### 宣传图检查
- [ ] 品牌色使用正确
- [ ] 文字清晰可读
- [ ] 重要信息突出
- [ ] 符合尺寸要求
- [ ] PNG格式 (透明背景如需)
