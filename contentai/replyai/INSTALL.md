# ReplyAI 安装指南

## 🚀 安装方式

### 方式一: Chrome Web Store 安装 (推荐)

1. 访问 Chrome Web Store 中的 ReplyAI 页面
2. 点击 "添加至 Chrome"
3. 确认权限请求
4. 安装完成！

### 方式二: 开发者模式安装

适用于测试或 Web Store 版本尚未发布时。

#### 步骤:

1. **下载扩展文件**
   ```bash
   # 下载 replyai-extension.zip
   # 解压到任意文件夹，例如 ~/replyai/
   ```

2. **打开Chrome扩展页面**
   - 在Chrome地址栏输入: `chrome://extensions/`
   - 或点击菜单 → 更多工具 → 扩展程序

3. **开启开发者模式**
   - 在扩展页面右上角，打开 "开发者模式" 开关

4. **加载扩展**
   - 点击 "加载已解压的扩展程序"
   - 选择解压后的 `replyai` 文件夹
   - 扩展图标将出现在工具栏

5. **固定扩展图标** (可选)
   - 点击Chrome工具栏的拼图图标
   - 找到 ReplyAI，点击图钉图标固定

## ⚙️ 首次配置

### 1. 获取 OpenAI API Key

1. 访问 [OpenAI Platform](https://platform.openai.com/api-keys)
2. 注册或登录您的OpenAI账号
3. 点击 "Create new secret key"
4. 复制生成的密钥 (以 `sk-` 开头)

**💡 提示:** 新用户通常有免费的API使用额度。

### 2. 配置扩展

1. 点击浏览器工具栏的 ReplyAI 图标
2. 在 "OpenAI API Key" 输入框粘贴您的密钥
3. 点击 "保存API Key"
4. 看到 "✅ API Key 已安全保存" 即配置成功！

### 3. 选择默认风格 (可选)

在设置页面可以选择默认回复风格：
- 👔 专业正式
- 😊 友好亲切  
- ⚡ 简洁明了
- ✨ 自定义

## 📧 使用指南

### 生成邮件回复

1. **打开邮件**
   - 在Gmail或Outlook中打开任意邮件

2. **读取邮件**
   - 点击ReplyAI图标
   - 点击 "📥 读取当前邮件"
   - 确认邮件内容显示正确

3. **选择风格**
   - 选择适合的回复风格
   - 或使用自定义风格描述

4. **生成回复**
   - 点击 "✨ 生成智能回复"
   - 等待AI生成 (通常1-3秒)

5. **使用回复**
   - 📤 **插入到邮件** - 自动填入回复框
   - 📋 **复制** - 复制到剪贴板手动粘贴

### 使用快捷模板

点击任意模板标签快速生成：
- 🙏 感谢 - 表达感谢
- ✅ 确认 - 确认收到
- 😔 道歉 - 表达歉意
- 🔄 跟进 - 跟进事项
- ❌ 拒绝 - 礼貌拒绝

## 🔧 常见问题

### Q: 为什么提示"请在Gmail或Outlook页面使用"?
A: ReplyAI目前仅支持 mail.google.com 和 outlook.* 域名。请确保您在正确的邮件页面。

### Q: 为什么无法读取邮件内容?
A: 请确保:
1. 已打开具体的邮件 (不是邮件列表页)
2. 页面已完全加载
3. 刷新页面后重试

### Q: 生成的回复不符合预期?
A: 尝试:
1. 使用"自定义"风格描述您想要的效果
2. 提供更多的风格描述细节
3. 手动修改生成的回复

### Q: 如何更换API Key?
A: 在扩展popup中输入新的API Key并保存即可，旧Key会被覆盖。

### Q: 如何清除所有数据?
A: 
1. 右键点击扩展图标 → 选项
2. 在设置页面点击 "🗑️ 清除所有本地数据"
3. 或卸载扩展
