## 技术栈

- **后端**: Cloudflare Workers (Edge Computing)
- **数据库**: Cloudflare D1 (SQLite)
- **AI**: Moonshot API (Kimi)
- **支付**: Coinbase Commerce + NowPayments
- **邮件**: Resend
- **语言**: JavaScript (ES Modules)

## 优势

1. **零服务器成本**: Cloudflare Workers 免费额度足够起步
2. **免实名**: 加密货币支付 + 无域名实名
3. **全球 CDN**: 自动全球加速
4. **全自动化**: 从获客到交付完全无人值守
5. **隐私友好**: 客户只需提供邮箱，无需注册账号

## 快速开始

```bash
cd /root/.openclaw/workspace/contentai/auto-system

# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填入 API 密钥

# 3. 创建数据库
npx wrangler d1 create contentai-db
# 复制 database_id 到 wrangler.toml

# 4. 初始化数据库
npx wrangler d1 execute contentai-db --file=./scripts/schema.sql

# 5. 部署
npx wrangler deploy
```
