### 技巧1: 用中文写思路，但给具体例子

❌ 不太好的输入：
"写一篇关于远程工作的博客"

✅ 更好的输入：
"写一篇关于远程工作的博客。我的观点：1) 异步沟通比会议更高效 2) 结果导向比打卡重要。目标读者是刚开始远程工作的管理者。"

**为什么有效：** AI需要上下文来理解你的视角

---

### 技巧2: 先生成大纲，再扩展内容

不要一次生成2000字。

更好的流程：
1. 先生成大纲（选"带大纲的博客"模板）
2. 检查大纲是否符合你的思路
3. 逐节生成内容

这样每个部分都能针对性调整，整体质量更高。

---

### 技巧3: 用"语气调整"功能匹配品牌调性

同样的内容，不同的语气：
- 专业正式 → 适合B2B白皮書
- 轻松友好 → 适合Twitter/LinkedIn
- 直接犀利 → 适合观点文

点击生成后的"调整语气"按钮试试看 👉 [去试试]({{app_url}}/dashboard)

### Resend 快速集成

```typescript
// lib/email.ts
import { Resend } from 'resend';

const resend = new Resend(process.env.RESEND_API_KEY);

export async function sendWelcomeEmail(email: string, name: string) {
  await resend.emails.send({
    from: 'Kimi @ ContentAI <kimi@contentai.app>',
    to: email,
    subject: '🎉 欢迎加入！你的第一篇AI内容准备好了吗？',
    html: welcomeEmailTemplate({ name }),
  });
}

// 邮件模板
function welcomeEmailTemplate({ name }: { name: string }) {
  return `
    <div style="font-family: Arial, sans-serif; max-width: 600px; margin: 0 auto;">
      <h1>欢迎加入 ContentAI！🎉</h1>
      <p>嗨 ${name || '朋友'},</p>
      <p>我是 Kimi，ContentAI 的创始人...</p>
      <!-- 完整邮件内容 -->
    </div>
  `;
}
```
