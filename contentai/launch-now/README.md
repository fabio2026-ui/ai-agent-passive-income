### 1. 社交货币（Social Currency）
**应用:** "转发给需要的人" / "分享给你认识的创业者"
- 人们喜欢分享"独家"或"有用"的信息
- 让用户感觉自己帮了别人

### 2. 互惠（Reciprocity）
**应用:** "免费是因为我曾经穷过，现在想帮人"
- 故事触发情感共鸣
- 用户感觉"欠你"，更愿意尝试和分享

**应用:** 创始人故事、真实用户案例

- "我也曾经是付不起订阅费的创业者"
