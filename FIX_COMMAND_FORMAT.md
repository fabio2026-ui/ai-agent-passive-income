# 🔧 修正命令格式

========================================
❌ 命令格式错误
========================================

你执行了：
```bash
sleep 3 python3 ~/ai_secure_v2.py &
```

错误：`sleep: invalid time interval: python3`

========================================
✅ 正确格式
========================================

**分开执行，或者加分号：**

```bash
# 方法1：分号分隔
pkill -9 Python; sleep 3; python3 ~/ai_secure_v2.py &

# 方法2：分开执行
pkill -9 Python
sleep 3
python3 ~/ai_secure_v2.py &
```

========================================
⚡ 现在执行
========================================

**一行命令（复制粘贴）：**

```bash
pkill -9 Python && sleep 3 && python3 ~/ai_secure_v2.py &
```

**或者分步：**

```bash
# 第一步：杀死Python
pkill -9 Python

# 第二步：等3秒
sleep 3

# 第三步：启动服务
python3 ~/ai_secure_v2.py &
```
