# 🔧 Chrome扩展快速安装指南

========================================
⚡ 最快安装方式 (2分钟)
========================================

## 方式1: 直接链接安装 (最快)

### 步骤1: 打开Chrome
点击Dock上的Chrome图标

### 步骤2: 直接访问扩展页面
在地址栏粘贴：
```
chrome.google.com/webstore/search/OpenClaw%20Browser%20Relay
```

或完整URL：
```
https://chrome.google.com/webstore/search/OpenClaw%20Browser%20Relay
```

### 步骤3: 找到扩展
搜索结果中找到 **"OpenClaw Browser Relay"**

图标样子：🦞 或类似 claws 的图标

### 步骤4: 安装
点击 **"添加到Chrome"** 按钮

### 步骤5: 确认
弹出窗口点击 **"添加扩展程序"**

========================================
🔍 如果搜索不到
========================================

## 方式2: 通过OpenClaw官网

### 步骤1: 访问OpenClaw文档
```
https://docs.openclaw.ai
```

### 步骤2: 查找扩展链接
搜索 "Browser Relay" 或 "Chrome Extension"

### 步骤3: 点击安装链接
通常有直接跳转到Chrome Web Store的链接

========================================
🎯 安装后连接
========================================

## 步骤1: 打开要控制的网页
例如：
```
https://www.fiverr.com
```

## 步骤2: 点击扩展图标
- 在Chrome工具栏找到新安装的扩展图标
- 通常在地址栏右边
- 图标可能是 🦞 或 ⬡

## 步骤3: 连接
点击 **"Attach Tab"** 或 **"Connect"**

## 步骤4: 确认连接成功
- 图标变绿色
- 或显示 "ON" / "Connected"
