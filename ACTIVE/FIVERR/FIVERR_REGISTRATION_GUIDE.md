# 🚀 Fiverr注册完整指南 - 复制粘贴即可上线

========================================
📍 你现在在哪里？
========================================

打开浏览器，访问: https://www.fiverr.com

**确认你已经完成账号注册（邮箱验证通过）**

如果是，继续下一步 👇

========================================
📝 第一步: 完善Profile (5分钟)
========================================

## 操作步骤:

1. 点击右上角头像 → **"Profile"**

2. 点击 **"Edit Description"**

3. 打开终端，复制bio.txt内容:
```bash
cat ~/ai-empire/launch/bio.txt | pbcopy  # Mac
# 或
cat ~/ai-empire/launch/bio.txt           # 然后手动复制
```

4. 粘贴到Fiverr的Description框

5. **添加技能标签** (点击Add Skills):
   - Video Editing
   - Social Media Marketing
   - Content Creation
   - TikTok Marketing
   - Instagram Marketing

6. 点击 **"Save"**

✅ Profile完成！

========================================
🎯 第二步: 创建Gig (15分钟)
========================================

## 操作步骤:

### 2.1 开始创建
1. 点击顶部 **"Selling"** → **"Gigs"**
2. 点击 **"Create A New Gig"**

### 2.2 填写Gig标题
1. 复制title.txt内容:
```bash
cat ~/ai-empire/launch/title.txt | pbcopy
```

2. 粘贴到 **"Gig Title"** 框

3. 选择类别:
   - **Category**: Video & Animation
   - **Subcategory**: Video Editing

4. 点击 **"Save & Continue"**

### 2.3 设置价格和套餐

**Scope & Pricing页面:**

**Basic Package ($50):**
- Name: "Starter - 1 Viral Video"
- Description: "1 AI-optimized video, 24h delivery"
- Delivery Time: 1 Day
- Revisions: 2
- Price: $50

**Standard Package ($100) - 勾选"Yes"启用:**
- Name: "Growth - 3 Videos + Strategy"
- Description: "3 viral videos + content strategy"
- Delivery Time: 2 Days
- Revisions: 3
- Price: $100

**Premium Package ($200) - 勾选"Yes"启用:**
- Name: "Viral - Full Growth Partner"
- Description: "5 videos + strategy + priority support"
- Delivery Time: 3 Days
- Revisions: Unlimited
- Price: $200

点击 **"Save & Continue"**

### 2.4 填写描述和FAQ

**Description部分:**
1. 复制description.txt:
```bash
cat ~/ai-empire/launch/description.txt | pbcopy
```

2. 粘贴到Description框

**FAQ部分:**
1. 点击 **"+ Add FAQ"** 5次
2. 复制faq.txt内容，逐个添加:
```bash
cat ~/ai-empire/launch/faq.txt
```

格式:
- Q: What do I need to provide?
- A: Just send me your raw video footage...

点击 **"Save & Continue"**

### 2.5 添加要求

在 **"Requirements"** 部分:

点击 **"+ Add Question"**:

**问题1:**
- Question: "Please share your video footage or clips"
- Answer Type: Free Text (Optional)

**问题2:**
- Question: "What platform is this for? (TikTok/Reels/Shorts)"
- Answer Type: Free Text (Optional)

**问题3:**
- Question: "Any specific style or reference videos?"
- Answer Type: Free Text (Optional)

点击 **"Save & Continue"**

### 2.6 上传样片 (Gallery)

**你有3个选项:**

**选项A: 使用截图 (推荐，最快)**
1. 打开素材视频: `~/ai-empire/launch/assets/`
2. 播放视频，截取3张精彩画面
3. 上传到Gallery
4. 添加描述:
   - "Product showcase video sample"
   - "Educational content sample"
   - "Lifestyle video sample"

**选项B: 使用现有视频**
1. 直接上传assets文件夹中的视频
2. 选择3个不同的

**选项C: 渲染新样片**
1. 运行: `bash ~/ai-empire/scripts/video-renderer.sh`
2. 上传生成的portfolio视频

点击 **"Save & Continue"**

### 2.7 发布Gig

1. 检查所有信息
2. 点击 **"Publish Gig"**
3. 🎉 完成！

========================================
✅ 第三步: 验证上线 (2分钟)
========================================

1. 进入Selling → Gigs
2. 点击你的Gig
3. 复制浏览器地址栏的链接
4. 发给我！

格式:
```

```
