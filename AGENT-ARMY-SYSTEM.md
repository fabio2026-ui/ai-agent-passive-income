## 🔄 Agent通信协议

### 消息格式
```json
{
  "from": "agent-name",
  "to": "target-agent or 'broadcast'",
  "type": "status|request|response|alert",
  "payload": {},
  "timestamp": "ISO-8601"
}
```

### 状态汇报频率
- **开发Agents**: 每30分钟汇报进度
- **监控Bots**: 每小时汇报监控数据
- **协调器**: 实时汇总，异常立即告警
