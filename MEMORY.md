## 五、核心学习原则 (2026-04-03更新)

### 技能获取黄金法则
> **碰到解决不了的情况 → 上网搜索新技能 → 没有现成技能 → 自己创造一个技能**

**执行流程**:
1. 遇到问题无法解决
2. 立即搜索是否有现成技能/工具
3. 如果存在 → 学习并应用
4. 如果不存在 → 自己创造一个技能
5. 将新技能标准化，供未来复用

**案例**:
- 需要临时邮箱 → 搜索/创建 temp-mail skill
- 需要视频生成 → 搜索/创建 jianying-video skill
- 需要竞品监控 → 创建自动化监控脚本

**原则**: 永远不被"我不会"卡住，要么学，要么造。
