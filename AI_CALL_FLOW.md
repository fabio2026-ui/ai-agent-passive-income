## ⚡ 一次完整调用的耗时

```
你发送指令 → 我构建命令:     0.5秒
构建命令 → 到达Tailscale:     0.3秒  (跨国网络)
Tailscale → Mac本地:          0.2秒
本地执行 → AI工厂响应:        2.0秒  (取决于任务复杂度)
AI工厂 → 返回给我:            0.5秒
我解析 → 回复你:              0.5秒
─────────────────────────────────────
总计:                         ~4秒
```

## 🎯 实际调用示例（真实发生）

### 示例1：生成视频
```
你："生成一个测试视频"

我发送：
curl -X POST https://fabiomacbook-air.tail284095.ts.net \
  -d '{"c":"say -v Ting-Ting '测试' -o voice.aiff && \
         ffmpeg ... 生成视频 ..."}'

3. 保存到Desktop

我回复你：

```
