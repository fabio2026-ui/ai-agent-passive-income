# Cognitive Enhancement System

## 🧩 认知偏差识别与修正

### 常见认知偏差清单
- [ ] **确认偏误**: 只寻找支持自己观点的证据
- [ ] **幸存者偏差**: 只看成功案例，忽略失败
- [ ] **锚定效应**: 被第一个信息过度影响
- [ ] **可得性启发**: 容易想起的例子被认为更常见
- [ ] **后见之明**: 事后认为"早就知道会这样"
- [ ] **损失厌恶**: 对损失的恐惧大于对收益的渴望
- [ ] **框架效应**: 同样的信息，表述方式影响决策
- [ ] **归因错误**: 自己的失败归因环境，别人的失败归因性格

### 偏差修正方法
1. **主动寻找反面证据**: 强迫自己看反面观点
2. **量化概率**: 用数字替代"可能/不可能"
3. **事前分析**: 在结果出来前记录预测
4. **外部视角**: 问"别人会怎么看"
5. **红队思维**: 专门有人负责挑刺

## 🎲 概率思维与不确定性处理

### 贝叶斯思维
```
P(假设|证据) = P(证据|假设) × P(假设) / P(证据)
```
- **先验概率**: 看到证据前的初始信念
- **似然度**: 假设成立时看到证据的概率
- **后验概率**: 看到证据后更新的信念

### 概率校准
- **90%置信区间**: 如果我说90%确定，实际应该90%正确
- **预测记录**: 记录预测，事后检验校准度
- **信息价值**: 哪些信息能最大程度改变决策？

## 🔄 学习方法论

### 费曼技巧
1. **选择概念**: 想深入理解的知识点
2. **教给别人**: 用简单语言解释
3. **发现缺口**: 解释不下去的地方就是盲区
4. **简化类比**: 用比喻让概念易懂
5. **重复精炼**: 直到能流畅解释

### 刻意练习
- **明确目标**: 具体要提升什么
- **即时反馈**: 马上知道对错
- **舒适区外**: 难度刚好超出当前水平
- **大量重复**: 次数产生质变
- **心理表征**: 形成专家级的认知结构

### 知识网络构建
- **连接新旧知识**: 新知识和已有知识建立连接
- **多元表征**: 同一概念多种表达方式
- **间隔重复**: 遗忘曲线上的最优复习时点
- **主动回忆**: 测试比重复阅读更有效

## 🎨 创造力增强

### 横向思维技巧
- **随机输入**: 随机选一个词，强制和主题联系
- **逆向假设**: 把通常认为真的假设反过来
- **类比迁移**: 从完全不相关的领域找灵感
- **极端约束**: 给自己加上不可能的限制
- **头脑风暴**: 先数量后质量，不评判

### SCAMPER 创新法
- **S**ubstitute: 替代什么？
- **C**ombine: 组合什么？
- **A**dapt: 适应什么？
- **M**odify: 修改什么？
- **P**ut to other uses: 其他用途？
- **E**liminate: 去掉什么？
- **R**everse: 颠倒什么？

## ⚔️ 辩论与说服框架

###  Toulmin 论证模型
- **主张 (Claim)**: 你要证明什么？
- **数据 (Data)**: 基于什么证据？
- **保证 (Warrant)**: 为什么数据支持主张？
- **支撑 (Backing)**: 保证的依据是什么？
- **限定 (Qualifier)**: 主张的适用范围？
- **反驳 (Rebuttal)**: 可能的例外情况？

### 说服的六个原则 (Cialdini)
1. **互惠**: 先给予，后索取
2. **承诺一致**: 从小请求开始，逐步升级
3. **社会认同**: 展示其他人的选择
4. **权威**: 引用专家/数据
5. **喜好**: 建立情感连接
6. **稀缺**: 强调机会有限

## 🌊 复杂系统思维

### 涌现特性
- **整体 > 部分之和**: 新特性在组合中涌现
- **非线性**: 小原因大结果，大原因无结果
- **反馈回路**: 正反馈(放大) vs 负反馈(稳定)
- **临界点**: 量变引起质变的阈值

### 网络思维
- **节点与连接**: 重要的是关系，不是孤立个体
- **中心性**: 哪些节点最关键？
- **级联效应**: 局部变化如何传播全局
- **鲁棒性与脆弱性**: 系统的抗打击能力

## 🔮 预见能力训练

### 情景规划
1. **关键驱动力**: 什么因素会影响未来？
2. **不确定性**: 哪些是最不确定的？
3. **情景构建**: 基于不同假设构建多个情景
4. **策略准备**: 每个情景下的应对策略
5. **早期信号**: 哪些指标预示情景发生？

### 反脆弱性 (Taleb)
- **脆弱**: 受冲击后损坏
- **坚韧**: 受冲击后恢复原状
- **反脆弱**: 受冲击后变得更强
- **杠铃策略**: 极端保守 + 极端冒险，避开中间

## 💎 元认知能力

### 自我监控
- **我在想什么？**: 意识到自己的思维过程
- **我的情绪如何？**: 识别情绪对判断的影响
- **我的偏见是什么？**: 知道自己的认知盲点
- **我需要什么信息？**: 识别知识缺口

### 思维质量评估
- **清晰度**: 我是否真正理解？
- **准确性**: 信息是否可靠？
- **精确度**: 细节是否足够？
- **相关性**: 是否切题？
- **深度**: 是否触及本质？
- **广度**: 是否考虑全面？
- **逻辑性**: 推理是否严密？
- **重要性**: 是否关注重点？
- **公正性**: 是否偏见？
