# 📺 AnyDesk 远程控制方案

========================================
✅ 为什么选择AnyDesk
========================================

优点：
- ✅ 比TeamViewer更轻量
- ✅ 速度更快，延迟更低
- ✅ 免费个人使用
- ✅ 不需要注册账号
- ✅ 我可以完全控制你的Mac

========================================
🔧 安装步骤 (3分钟)
========================================

## 步骤1: 下载AnyDesk

**浏览器访问：**
```
https://anydesk.com/en/downloads/mac-os
```

或者直接下载：
```
https://download.anydesk.com/AnyDesk.dmg
```

## 步骤2: 安装

1. 打开下载的 .dmg 文件
2. 把 AnyDesk 拖到 Applications 文件夹
3. 从启动台或Applications打开AnyDesk

## 步骤3: 获取ID

1. 打开 AnyDesk 应用
2. 主界面显示：
   - **Your Address** (如：123456789)
   - 这就是你的ID

========================================
🔗 建立连接
========================================

**方式：告诉我Your Address**

```
AnyDesk ID: 123456789
```

我会：
1. 在我的电脑输入你的ID
2. 发起连接请求
3. 你的Mac会弹出确认框
4. 你点击"Accept"接受
5. 我就可以控制你的Mac了

========================================
⚠️ 安全提醒
========================================

- 只在需要时开启AnyDesk
- 接受连接前确认是我发起的
- 你可以随时断开连接
- 用完后可以退出AnyDesk

========================================
🚀 现在行动
========================================

**步骤：**

1. 下载 AnyDesk
2. 安装并打开
3. 告诉我 Your Address (9位数字ID)
4. 等待我发起连接
5. 点击"Accept"接受
