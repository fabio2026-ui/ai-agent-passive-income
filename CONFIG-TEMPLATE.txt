# 🔐 AI Empire 配置文件模板
**文件名**: `ai-empire-config.txt`
**用途**: 安全传输API密钥和Token
**填写说明**: 将`[你的...]`替换为实际值，保持格式不变

---

## 1️⃣ Cloudflare 配置（部署API服务）

### 方式A: API Token（推荐）
```
CLOUDFLARE_API_TOKEN=[你的Cloudflare API Token]
```

**如何获取**:
1. 访问 https://dash.cloudflare.com/profile/api-tokens
2. 点击 "Create Token"
3. 选择模板 "Edit Cloudflare Workers"
4. 选择你的域名区域 (如: eucrossborder.com)
5. 点击 "Continue" → "Create Token"
6. 复制生成的Token（长字符串）

### 方式B: Global API Key（备用）
```
CLOUDFLARE_EMAIL=[你的Cloudflare注册邮箱]
CLOUDFLARE_GLOBAL_API_KEY=[你的Global API Key]
```

**如何获取**:
1. 访问 https://dash.cloudflare.com/profile/api-tokens
2. 拉到页面底部 "Global API Key"
3. 点击 "View"，验证密码后复制

---

## 2️⃣ Stripe 配置（收款功能）

### 生产环境密钥（live）
```
STRIPE_PUBLISHABLE_KEY_LIVE=[你的pk_live_...]
STRIPE_SECRET_KEY_LIVE=[你的sk_live_...]
STRIPE_WEBHOOK_SECRET_LIVE=[你的whsec_...] (可选，用于webhook)
```

### 测试环境密钥（test）
```
STRIPE_PUBLISHABLE_KEY_TEST=[你的pk_test_...]
STRIPE_SECRET_KEY_TEST=[你的sk_test_...]
```

**如何获取**:
1. 访问 https://dashboard.stripe.com/apikeys
2. 确保切换到 "Live mode"（测试用Test mode）
3. 复制 Publishable key 和 Secret key

---

## 3️⃣ Chrome Web Store 配置（上架扩展）

```
CHROME_CLIENT_ID=[你的Client ID]
CHROME_CLIENT_SECRET=[你的Client Secret]
CHROME_REFRESH_TOKEN=[你的Refresh Token]
```

**如何获取**:
1. 访问 https://chrome.google.com/webstore/devconsole
2. 注册开发者账号（支付$5）
3. 进入 "设置" → "开发者账号"
4. 按照 https://developer.chrome.com/docs/webstore/using-api 获取OAuth凭证

---

## 4️⃣ 域名配置（已购买域名）

```
DOMAIN_EU=[你的欧盟域名，如: eucrossborder.com]
DOMAIN_UK=[你的英国域名，如: ukcrossborder.com]
DOMAIN_US=[你的美国域名，如: ustaxhelper.com]
DOMAIN_CA=[你的加拿大域名，如: cataxhelper.com]
```

---

## 5️⃣ Reddit 配置（自动发帖）

```
REDDIT_USERNAME=[你的Reddit用户名]
REDDIT_PASSWORD=[你的Reddit密码]
REDDIT_CLIENT_ID=[你的App Client ID]
REDDIT_CLIENT_SECRET=[你的App Client Secret]
```

**如何获取**:
1. 访问 https://www.reddit.com/prefs/apps
2. 点击 "create another app..."
3. 选择 "script" 类型
4. 复制 Client ID 和 Client Secret

---

## 6️⃣ Twitter/X 配置（可选）

```
TWITTER_API_KEY=[你的API Key]
TWITTER_API_SECRET=[你的API Secret]
TWITTER_ACCESS_TOKEN=[你的Access Token]
TWITTER_ACCESS_TOKEN_SECRET=[你的Access Token Secret]
```

**如何获取**:
1. 访问 https://developer.twitter.com/en/portal/dashboard
2. 创建 Project → App
3. 生成 Access Token
