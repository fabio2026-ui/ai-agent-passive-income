# Product Hunt 发布执行清单

## 发布前24小时
- [ ] 创建Product Hunt账号 (用Twitter/Google登录)
- [ ] 完善个人资料 (头像、Bio、Maker身份)
- [ ] 邀请团队成员作为Hunter

## 发布前1小时
- [ ] 准备首评内容
- [ ] 设置Twitter/Facebook分享
- [ ] 通知网络 (邮件列表、Discord、Slack)

## 发布时刻 (周二 9AM PST)
- [ ] 点击 "Post a Product"
- [ ] 填写所有字段
- [ ] 发布首评
- [ ] 分享到社交媒体

## 发布后第1小时 (黄金时间)
- [ ] 回复每条评论 (目标: 50+评论)
- [ ] 感谢支持者
- [ ] 回答技术问题
- [ ] 引导到网站注册

## 发布后全天
- [ ] 每30分钟检查一次
- [ ] 持续回复评论
- [ ] 更新产品信息
- [ ] 记录数据
