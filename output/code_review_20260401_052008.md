### 1. SQL注入漏洞 [C-001]

**位置**: 
- `bad_code_example.py:78` - `get_user_by_name()` 方法
- `database_module.py:34` - `get_user_by_name()` 方法
- `example_flask_api.py:48` - `create_user()` 函数

- `example_flask_api.py:155` - `login()` 函数

**问题描述**:

```python
# 危险代码示例

```

**风险**: 攻击者可以通过构造特殊输入来执行任意 SQL 命令，包括删除数据库、窃取数据等。

**修复建议**:
```python

query = "SELECT * FROM users WHERE name = ?"

```

### 3. 使用不安全的MD5哈希 [C-003]

**位置**:
- `bad_code_example.py:32` - `addUser()` 方法
- `bad_code_example.py:53` - `login()` 方法
- `example_flask_api.py:52` - 密码哈希
- `example_flask_api.py:153` - 登录验证

**问题描述**:
使用 MD5 算法对密码进行哈希，MD5 已被证明不安全，容易被暴力破解。

```python
hash=hashlib.md5(pwd.encode()).hexdigest()
```

**风险**: MD5 哈希可以被彩虹表快速破解，用户密码容易被泄露。

**修复建议**:
```python
# 使用 bcrypt
import bcrypt
hashed = bcrypt.hashpw(password.encode('utf-8'), bcrypt.gensalt())
# 验证时
bcrypt.checkpw(password.encode('utf-8'), hashed)
```

**优先级**: 🔴 Critical | **修复时间**: 立即

### 6. 危险的 os.system() 调用 [C-006]

**位置**:
- `bad_code_example.py:81` - `backup_database()` 函数

**问题描述**:
使用 `os.system()` 执行系统命令，且命令中包含时间戳拼接，可能被命令注入。

```python
cmd='cp users.json backup_'+str(int(time.time()))+'.json'
os.system(cmd)
```

**风险**: 如果 `time.time()` 被污染（虽然本例不太可能被攻击），或后续修改中引入变量，可能导致命令注入。

**修复建议**:
```python
import shutil
from datetime import datetime
backup_path = f'backup_{datetime.now():%Y%m%d_%H%M%S}.json'
shutil.copy('users.json', backup_path)
```

### 7. 缺乏身份验证和授权 [C-007]

**位置**:

- `example_flask_api.py:89` - `update_user()` 函数


**风险**: 未授权访问，数据被恶意修改或删除。

**修复建议**:
```python
from functools import wraps
from flask import request, g

def require_auth(f):
    @wraps(f)
    def decorated(*args, **kwargs):
        token = request.headers.get('Authorization')
        if not token:
            return jsonify({'error': 'Unauthorized'}), 401
        # 验证 token 并设置 g.user
        return f(*args, **kwargs)
    return decorated

def require_owner(f):
    @wraps(f)
    def decorated(*args, **kwargs):
        user_id = kwargs.get('id')
        if g.user.id != user_id and not g.user.is_admin:
            return jsonify({'error': 'Forbidden'}), 403
        return f(*args, **kwargs)
    return decorated

@app.route('/users/<id>', methods=['PUT'])
@require_auth
@require_owner
def update_user(id):
    ...
```

### H-002: 性能问题 - O(n²) 算法

**位置**: `bad_code_example.py:61-68` - `process_data()` 函数

**问题描述**:
```python
def process_data(data_list):
    result=[]
    for i in range(len(data_list)):
        for j in range(len(data_list)):
            if i!=j:
                if data_list[i]==data_list[j]:
                    result.append(data_list[i])
    return result
```

**风险**: 对于大数据集会严重影响性能。

**修复建议**:
```python
def process_data(data_list):
    seen = set()
    duplicates = set()
    for item in data_list:
        if item in seen:
            duplicates.add(item)
        seen.add(item)
    return list(duplicates)
```
