### 7. 竞态条件与文件操作问题 [HIGH]

**位置:** `bad_code_example.py:59-61`, `first-sale-monitor.py:47-52`

**问题代码:**
```python
# bad_code_example.py
def save(self):
    with open(self.db,'w') as f:
        json.dump(self.users,f)

# 多进程/线程环境下可能损坏数据
```

**修复建议:**
```python
import fcntl
import tempfile
import os

def save(self):
    # 使用临时文件 + 原子重命名
    fd, temp_path = tempfile.mkstemp(dir=os.path.dirname(self.db))
    try:
        with os.fdopen(fd, 'w') as f:
            # 文件锁防止并发写入
            fcntl.flock(f, fcntl.LOCK_EX)
            json.dump(self.users, f)
            fcntl.flock(f, fcntl.LOCK_UN)
        # 原子重命名
        os.rename(temp_path, self.db)
    except:
        os.unlink(temp_path)
        raise
```

### 9. O(n²) 性能问题 [HIGH]

**位置:** `bad_code_example.py:68-77`

**问题代码:**
```python
def process_data(data_list):
    """处理数据（性能问题示例）"""
    result=[]
    for i in range(len(data_list)):
        for j in range(len(data_list)):  # O(n²)
            if i!=j:
                if data_list[i]==data_list[j]:
                    result.append(data_list[i])
    return result
```

**优化方案:**
```python
from collections import Counter

def process_data(data_list):
    """O(n) 复杂度"""
    counts = Counter(data_list)
    return [item for item, count in counts.items() if count > 1]

# 或使用集合
import itertools

def process_data(data_list):
    """使用集合去重"""
    seen = set()
    duplicates = set()
    for item in data_list:
        if item in seen:
            duplicates.add(item)
        seen.add(item)
    return list(duplicates)
```

```python
@app.route('/users', methods=['GET'])

    # 缺乏分页，可能导致性能问题
```

```python
from flask import request

@app.route('/users', methods=['GET'])
def get_users():
    page = request.args.get('page', 1, type=int)
    per_page = min(request.args.get('per_page', 20, type=int), 100)
    
    offset = (page - 1) * per_page
    cursor = g.db.execute(
        "SELECT id, username, email, age, created_at FROM users LIMIT ? OFFSET ?",
        (per_page, offset)
    )
    users = cursor.fetchall()
    
    return jsonify({
        'data': [dict(user) for user in users],
        'page': page,
        'per_page': per_page
    })
```
