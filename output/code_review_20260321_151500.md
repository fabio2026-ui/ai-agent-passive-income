#### 2. 路径遍历风险
**文件**: `intel_hub.py`

```python
# 潜在风险
filepath = self.BASE_PATH / category / f"{key}.json"
```

如果 `category` 或 `key` 包含 `../` 等字符，可能导致路径遍历。

**修复建议**:
```python
import re
from pathlib import Path

def _sanitize_path_component(self, component: str) -> str:
    """清理路径组件，防止路径遍历攻击"""
    # 移除危险字符
    sanitized = re.sub(r'[^\w\-_.]', '_', component)
    # 阻止以点开头的隐藏文件
    if sanitized.startswith('.'):
        sanitized = '_' + sanitized
    return sanitized

def _get_safe_filepath(self, category: str, key: str) -> Path:
    safe_category = self._sanitize_path_component(category)
    safe_key = self._sanitize_path_component(key)
    return self.BASE_PATH / safe_category / f"{safe_key}.json"
```

## 三、性能优化建议

### 3.1 文件I/O优化

**文件**: `intel_hub.py`

当前每次读取都打开文件，可以添加内存缓存层:

```python
from functools import lru_cache
from cachetools import TTLCache

class IntelHub:
    def __init__(self):
        # 添加内存缓存，TTL为5分钟
        self._memory_cache = TTLCache(maxsize=1000, ttl=300)
    
    def get(self, category: str, key: str) -> Optional[Dict]:
        cache_key = f"{category}:{key}"
        
        # 先查内存缓存
        if cache_key in self._memory_cache:
            return self._memory_cache[cache_key]
        
        # 再查文件
        filepath = self.BASE_PATH / category / f"{key}.json"
        if not filepath.exists():
            return None
        
        with open(filepath, 'r', encoding='utf-8') as f:
            data = json.load(f)
        
        # 写入内存缓存
        self._memory_cache[cache_key] = data
        return data
```

### 3.4 减少重复计算

**文件**: `legion_hq.py`

```python
async def _main_loop(self):
    while self._running:
        # 问题：每次循环都重新计算
        health_score = await self._calculate_health_score()
        
# 建议：缓存健康分数，定期更新
```
