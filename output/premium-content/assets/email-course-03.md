## 第1天：建立选题库（2小时）

### 为什么选题库如此重要？

**没有选题库：**
- 每天花1小时想写什么
- 临时抱佛脚，质量不稳定
- 容易放弃

**有选题库：**
- 提前储备30-50个选题
- 随时开始创作，无压力
- 保持持续输出

### 如何批量生成选题？

**方法1：AI辅助选题**
```
Prompt：
"你是一位[你的领域]的内容专家。
请基于以下角度，生成20个内容选题：
1. 常见问题和痛点
2. 常见误区
3. 实用技巧和教程
4. 案例分析和故事
5. 行业趋势和洞察

每个选题要具体、有吸引力、有实用价值。"
```

**方法2：竞品分析**
- 找到5个同领域优秀账号
- 分析他们最火的内容
- 用自己的角度重新解读

**方法3：用户需求挖掘**
- 搜索相关问题（知乎/百度知道）
- 查看评论区高频问题
- 社群中的常见疑问

### 选题库模板

```
📁 选题库
├── 🔥 热点选题（时效性，1周内发布）
│   ├── [选题1] 预计发布时间：
│   └── [选题2] 预计发布时间：
├── 📚 常青选题（长期有效，随时可发）
│   ├── [选题1] 优先级：高/中/低
│   ├── [选题2] 优先级：高/中/低
│   └── ...（至少储备20个）
├── 💡 灵感收集（待完善）
│   ├── [碎片想法1]
│   └── [碎片想法2]
└── 📊 数据分析（基于数据优化）
    ├── 高互动选题类型：
    └── 待测试选题：
```

## 第2-3天：内容批量创作（4小时）

### 批量创作的魔力

**为什么要批量？**
- 进入心流状态，效率更高
- 减少任务切换的时间浪费
- 保持风格一致性

**我的批量创作日程：**
- 周二上午：写2篇长文（深度内容）
- 周二下午：做5条图文（社媒内容）
- 周三上午：录2个视频（如有需要）

### AI辅助写作工作流

**Step 1：大纲生成（10分钟/篇）**
```
Prompt：
"基于选题'[选题名称]'，生成文章大纲。
要求：
- 2000字左右
- 5-7个小节
- 每个小节包含核心观点
- 有案例或数据支撑
- 有 actionable takeaways"

AI输出 → 人工调整结构 → 确定大纲
```

**Step 2：分段撰写（30分钟/篇）**
```
Prompt：
"基于以下大纲，撰写[第一节]内容。
风格要求：
- 口语化，像和朋友聊天
- 每段不超过3行
- 使用具体例子
- 适当使用emoji

大纲：[粘贴小节大纲]"

AI输出 → 人工编辑 → 润色完善
```

**Step 3：优化迭代（20分钟/篇）**
```
优化指令：
- "让开头更有吸引力"
- "增加一个具体案例"
- "简化这段话的表达"
- "检查事实准确性"
```

**总时间：60分钟/篇 2000字文章**
（传统方式需要4-6小时）

### 社媒内容批量模板

**小红书批量生产：**
```
一次生成10条：

Prompt：
"基于主题'[主题]'，生成10条小红书文案。
要求：
- 每篇200-300字
- 使用emoji
- 有实用价值
- 引导互动

格式：
标题：
正文：
标签："

AI生成 → 选择最佳3条 → 微调 → 配图 → 完成
```

**时间：1小时产出10条小红书内容**

### 关键指标追踪

| 指标 | 工具 | 健康标准 |

| 阅读量 | 平台后台 | 稳定增长 |

### 优化循环

```
数据 → 分析 → 假设 → 测试 → 验证 → 迭代
```

1. 哪条内容表现最好？为什么？
