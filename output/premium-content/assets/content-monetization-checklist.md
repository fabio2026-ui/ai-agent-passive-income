# 内容变现检查清单
## ContentAI实战工具包

---

## 如何使用本清单

1. **打印出来**或保存在手机备忘录
2. **每次创作时**对照检查
3. **每周回顾**，找出改进点
4. **根据阶段选择**，不需要一次性全部完成

---

## 一、内容创作前检查

### 1.1 选题验证清单

| 检查项 | ✅ | 备注 |
|--------|-----|------|
| 目标受众是否明确？ | ☐ | 写下具体的人群画像 |
| 是否解决具体痛点？ | ☐ | 痛点越具体越好 |
| 是否有搜索需求？ | ☐ | 用关键词工具验证 |
| 竞争是否过于激烈？ | ☐ | 避开红海话题 |
| 是否在我的能力范围内？ | ☐ | 能输出专业内容 |
| 是否与我的定位一致？ | ☐ | 符合个人品牌 |
| 是否有变现潜力？ | ☐ | 商业相关度 |

**通过标准：** 至少5个✅才值得投入时间

### 1.2 研究阶段清单

| 检查项 | ✅ | 来源 |
|--------|-----|------|
| 已收集3-5个权威来源？ | ☐ | 学术论文/官方数据 |
| 已提取关键数据/案例？ | ☐ | 有具体数字支撑 |
| 已了解竞品内容？ | ☐ | 分析Top 3文章 |
| 已找到内容缺口？ | ☐ | 别人没讲到的点 |
| 已确定核心观点？ | ☐ | 一句话概括价值 |
| 已收集相关金句/引用？ | ☐ | 增强说服力 |

### 1.3 AI Prompt准备

| 检查项 | ✅ | 模板位置 |
|--------|-----|----------|
| 已选择合适角色？ | ☐ | 专家/助手/编辑 |
| 已明确输出格式？ | ☐ | 大纲/全文/要点 |
| 已提供足够背景？ | ☐ | 目标读者/场景 |
| 已设定语气风格？ | ☐ | 专业/轻松/幽默 |

## 二、内容创作中检查

### 2.1 文章结构检查

#### 标题检查
| 检查项 | ✅ | 优化建议 |
|--------|-----|----------|
| 是否包含关键词？ | ☐ | 前5个字出现 |
| 是否有数字？ | ☐ | "3个技巧"比"技巧"好 |
| 是否制造好奇/冲突？ | ☐ | 让人想点击 |
| 是否控制在20字以内？ | ☐ | 移动端完整显示 |
| 是否承诺具体价值？ | ☐ | 读者能得到什么 |

#### 开头检查（3秒法则）
| 检查项 | ✅ | 示例 |
|--------|-----|------|
| 是否直击痛点？ | ☐ | "你是否也遇到过..." |
| 是否建立共鸣？ | ☐ | 让读者点头 |
| 是否承诺价值？ | ☐ | "读完本文你将学会..." |
| 是否引发好奇？ | ☐ | "大多数人不知道..." |
| 是否在3秒内完成？ | ☐ | 快速进入主题 |

#### 正文结构检查
| 检查项 | ✅ | 标准 |
|--------|-----|------|
| 是否使用小标题？ | ☐ | 每500字一个 |
| 段落是否简短？ | ☐ | 不超过4行 |
| 是否有列表/表格？ | ☐ | 提升可读性 |
| 是否有过渡句？ | ☐ | 段落间衔接 |
| 是否有案例/数据？ | ☐ | 具体可信 |
| 是否有图片/视觉？ | ☐ | 每3-4段一张 |

#### 结尾检查
| 检查项 | ✅ | 作用 |
|--------|-----|------|
| 是否有总结？ | ☐ | 强化核心观点 |
| 是否有行动号召？ | ☐ | 告诉读者下一步 |
| 是否有互动引导？ | ☐ | 提问/评论 |
| 是否有相关推荐？ | ☐ | 延长阅读 |

### 2.2 AI内容质量检查

| 检查项 | ✅ | 处理方法 |
|--------|-----|----------|
| 事实是否正确？ | ☐ | 逐一核实 |
| 数据是否最新？ | ☐ | 确认年份和来源 |
| 观点是否独特？ | ☐ | 添加个人见解 |
| 语气是否一致？ | ☐ | 全文统一调整 |
| 是否有AI痕迹？ | ☐ | 重写机械表达 |
| 是否本土化？ | ☐ | 符合读者习惯 |
| 是否有过渡词堆砌？ | ☐ | 删除"此外""总之"等 |

## 三、内容发布前检查

### 3.1 SEO优化检查

| 检查项 | ✅ | 工具 |
|--------|-----|------|
| 标题含核心关键词？ | ☐ | - |
| 前100字出现关键词？ | ☐ | - |
| 关键词密度2-3%？ | ☐ | SEO插件 |
| H1/H2标签使用正确？ | ☐ | - |
| 内链3-5个？ | ☐ | 相关文章 |
| 外链1-2个权威站？ | ☐ | 学术/官方 |
| 图片alt标签填写？ | ☐ | 含关键词 |
| URL简洁含关键词？ | ☐ | - |
| 元描述已优化？ | ☐ | 155字以内 |

### 3.2 格式和排版检查

| 检查项 | ✅ | 标准 |
|--------|-----|------|
| 字体大小适中？ | ☐ | 手机阅读舒适 |
| 行间距足够？ | ☐ | 1.5-1.8倍 |
| 重点有加粗/颜色？ | ☐ | 但不要过多 |
| 无错别字？ | ☐ | 用工具检查 |
| 标点符号规范？ | ☐ | 中英文区分 |
| 图片清晰无水印？ | ☐ | 高清素材 |
| 移动端预览正常？ | ☐ | 必检查项 |

### 3.3 法律和安全检查

| 检查项 | ✅ | 注意 |
|--------|-----|------|
| 无敏感词/违规内容？ | ☐ | 用工具检测 |
| 引用已标注来源？ | ☐ | 避免抄袭 |
| 图片有版权或自制？ | ☐ | 避免侵权 |
| 数据有可靠来源？ | ☐ | 可追溯 |
| 无绝对化用语？ | ☐ | "第一""最佳"等 |
| 隐私信息已脱敏？ | ☐ | 保护个人信息 |

## 七、分阶段重点清单

### 阶段1：起步期（0-1000粉丝）

**重点：**
- [ ] 确定细分领域
- [ ] 建立内容节奏（每周3-5条）
- [ ] 学习基础工具
- [ ] 积累30条内容
- [ ] 加入相关社群

**不要做的事：**
- ☐ 急于变现
- ☐ 追求多平台
- ☐ 内容太杂
- ☐ 三天打鱼

### 阶段2：成长期（1000-10000粉丝）

**重点：**
- [ ] 分析爆款，复制成功
- [ ] 开始跨平台分发
- [ ] 建立内容模板库
- [ ] 申请联盟项目
- [ ] 尝试轻度变现

**不要做的事：**
- ☐ 为了涨粉牺牲质量
- ☐ 接不相关的广告
- ☐ 忽视老粉丝
- ☐ 停止学习

### 阶段3：变现期（10000+粉丝）

**重点：**
- [ ] 开发付费产品
- [ ] 建立自动化系统
- [ ] 扩大团队/外包
- [ ] 多元化收入
- [ ] 建立个人品牌

**不要做的事：**
- ☐ 过度商业化
- ☐ 忽视内容质量
- ☐ 单打独斗
- ☐ 停止创新

## 八、紧急问题处理清单

### 内容被抄袭怎么办？

- [ ] 截图保存证据
- [ ] 联系对方删除
- [ ] 向平台投诉
- [ ] 申请原创保护
- [ ] 公开发声（必要时）

### 账号被限流怎么办？

- [ ] 检查是否违规
- [ ] 停止可疑操作
- [ ] 联系客服申诉
- [ ] 降低发布频率
- [ ] 提高内容质量
- [ ] 等待恢复（通常7-30天）

### 负面评论怎么办？

- [ ] 冷静，不要情绪化
- [ ] 判断是否有理
- [ ] 有理：感谢并改进
- [ ] 无理：礼貌解释
- [ ] 恶意：删除/拉黑
- [ ] 不要纠缠

### 创作瓶颈怎么办？

- [ ] 暂时休息1-2天
- [ ] 阅读/学习充电
- [ ] 与同行交流
- [ ] 尝试新内容类型
- [ ] 回顾读者反馈
- [ ] 寻找新灵感来源
