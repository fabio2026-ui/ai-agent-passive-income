## 1. 项目概述

### 1.1 什么是RAGFlow

RAGFlow是一个领先的开源**检索增强生成(RAG)引擎**，融合了最先进的RAG技术与Agent能力，为LLM提供优质的上下文层。

### 1.2 核心特性

| 特性 | 描述 |
|------|------|
| 🍭 **Quality in, quality out** | 基于深度文档理解的知识提取 |
| 🍱 **Template-based chunking** | 智能且可解释的分块策略 |
| 🌱 **Grounded citations** | 可视化文本分块，减少幻觉 |
| 🍔 **多数据源兼容** | 支持Word、Excel、PDF、图片、网页等 |
| 🛀 **自动化RAG工作流** | 面向个人和企业的简化RAG流程 |

### 1.3 技术栈

- **后端**: Python 3.12+ (主服务), Go (高性能引擎)
- **前端**: React + TypeScript
- **存储**: Elasticsearch / Infinity (搜索引擎), MySQL (元数据), Redis (缓存)
- **对象存储**: MinIO
- **部署**: Docker, Docker Compose, Helm

### 3.3 RAG流程架构

```
文档上传 → 深度解析(DeepDoc) → 智能分块 → Embedding → 向量存储
                                           ↓
用户查询 → Query理解 → 混合检索 → Rerank → 上下文构建 → LLM生成 → 结果返回
                                           ↑
                                    引用溯源(Citation)
```

from typing import Literal
from enum import Enum

class LLMErrorType(Enum):
    RATE_LIMIT = "rate_limit"
    TIMEOUT = "timeout"
    AUTH = "auth"
    UNKNOWN = "unknown"

class LLMError(Exception):
    def __init__(self, message: str, error_type: LLMErrorType, retryable: bool):
        super().__init__(message)
        self.error_type = error_type
        self.retryable = retryable

try:
