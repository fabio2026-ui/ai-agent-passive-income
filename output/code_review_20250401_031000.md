## 🔴 严重问题（需立即修复）

### 1. SQL注入漏洞

**文件**: `database_module.py`

```python
# 危险代码
query="SELECT * FROM users WHERE name='"+username+"'"
cursor.execute(full_sql)  # 字符串拼接SQL
```

**风险**: 攻击者可通过构造恶意输入执行任意SQL命令，导致数据泄露、篡改或删除。

**修复建议**:
```python
# 正确做法 - 使用参数化查询
query = "SELECT * FROM users WHERE name = ?"
cursor.execute(query, (username,))
```

**优先级**: 🔴 P0 - 立即修复

### 2. 使用eval()执行任意代码

**文件**: `database_module.py`

```python
def parse_config(self, config_str):
    # 危险：使用eval
    return eval(config_str)
```

**风险**: eval()会执行传入的任意Python代码，攻击者可执行系统命令。

**修复建议**:
```python
import json
import ast

def parse_config(self, config_str):
    # 安全做法 - 使用json或ast.literal_eval
    return json.loads(config_str)
    # 或: return ast.literal_eval(config_str)
```

**优先级**: 🔴 P0 - 立即修复

### 3. 硬编码敏感信息

### 4. 不安全的密码哈希（MD5）

**文件**: `bad_code_example.py`

```python
# 问题：MD5已过时，不安全
hash=hashlib.md5(pwd.encode()).hexdigest()
```

**风险**: MD5已被破解，彩虹表攻击可在秒级破解密码。

**修复建议**:
```python
import bcrypt

# 正确做法
salt = bcrypt.gensalt()
hashed = bcrypt.hashpw(password.encode('utf-8'), salt)

# 验证
check = bcrypt.checkpw(password.encode('utf-8'), hashed)
```

**优先级**: 🔴 P0 - 立即修复

## 🟠 高危问题（强烈建议修复）

### 5. 命令注入风险

**文件**: `bad_code_example.py`

```python
def backup_database():
    cmd='cp users.json backup_'+str(int(time.time()))+'.json'
    os.system(cmd)
```

**风险**: os.system()执行shell命令，如果time.time()可被操控，存在注入风险。

**修复建议**:
```python
import shutil
import subprocess

def backup_database():
    backup_path = f'backup_{int(time.time())}.json'
    shutil.copy('users.json', backup_path)
```

## 🟡 性能问题

### 11. O(n²)算法复杂度

**文件**: `bad_code_example.py`

```python
def process_data(data_list):
    result=[]
    for i in range(len(data_list)):
        for j in range(len(data_list)):
            if i!=j:
                if data_list[i]==data_list[j]:
                    result.append(data_list[i])
    return result
```

**优化建议**:
```python
def process_data(data_list):
    seen = set()
    duplicates = set()
    for item in data_list:
        if item in seen:
            duplicates.add(item)
        seen.add(item)
    return list(duplicates)
# 复杂度从O(n²)降到O(n)
```

    ...
