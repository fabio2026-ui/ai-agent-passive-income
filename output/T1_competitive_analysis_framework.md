## 1. 竞品分析方法论

### 1.1 波特五力分析模型 (Porter's Five Forces)

**适用场景**: 评估行业竞争强度和盈利能力

```
┌─────────────────────────────────────────────────────────────────┐
│                    新进入者威胁 (Threat of New Entrants)            │
│                              ↓                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                         │   │
│  │    供应商议价能力      【行业竞争】      买方议价能力      │   │
│  │   (Bargaining Power)   (Rivalry)     (Bargaining Power)  │   │
│  │        ↑                   ↓                ↑            │   │
│  │                                                         │   │
│  └─────────────────────────────────────────────────────────┘   │
│                              ↓                                  │
│                  替代品威胁 (Threat of Substitutes)              │
└─────────────────────────────────────────────────────────────────┘
```

**分析框架模板**:

| 力量维度 | 高威胁指标 | 低威胁指标 | 评估问题清单 |
|---------|-----------|-----------|-------------|
| **新进入者威胁** | 低进入壁垒、高利润率、品牌不重要 | 高进入壁垒、规模经济、监管严格 | 1. 进入需要多少资本?<br>2. 现有玩家的规模优势多大?<br>3. 监管壁垒高吗? |
| **供应商议价能力** | 供应商集中、转换成本高、无替代品 | 供应商分散、标准化产品、转换成本低 | 1. 供应商集中度?<br>2. 我们的采购占比?<br>3. 替代供应商多吗? |
| **买方议价能力** | 买方集中、标准化产品、转换成本低 | 买方分散、定制化产品、高转换成本 | 1. 大客户占比?<br>2. 客户获取成本?<br>3. 替代选择多吗? |
| **替代品威胁** | 替代品性能好、价格低、转换容易 | 替代品性能差、价格高、转换困难 | 1. 有哪些完全不同的解决方案?<br>2. 技术变革风险?<br>3. 客户对价格的敏感度? |
| **行业内竞争** | 增长慢、退出壁垒高、产品同质化 | 增长快、差异化明显、退出容易 | 1. 竞争对手数量和规模?<br>2. 行业增长率?<br>3. 固定成本占比? |

**评分标准**:
- 🔴 高威胁 (3分) | 🟡 中等威胁 (2分) | 🟢 低威胁 (1分)
- **总分**: 5-7分 = 吸引力高 | 8-11分 = 中等 | 12-15分 = 吸引力低

### 1.2 SWOT 分析框架

**适用场景**: 评估企业/产品的内部优劣势和外部机会威胁

```
┌───────────────────────┬───────────────────────┐
│     S (优势)          │     W (劣势)          │
│    Strengths          │    Weaknesses         │
│                       │                       │
│ • 内部积极因素        │ • 内部消极因素        │
│ • 可控因素            │ • 可控因素            │
│ • 与竞品对比的优势    │ • 与竞品对比的劣势    │
├───────────────────────┼───────────────────────┤
│     O (机会)          │     T (威胁)          │
│    Opportunities      │    Threats            │
│                       │                       │
│ • 外部积极因素        │ • 外部消极因素        │
│ • 不可控但可利用      │ • 不可控且需应对      │
│ • 市场/技术/政策趋势  │ • 竞争/监管/经济风险  │
└───────────────────────┴───────────────────────┘
```

**SWOT 交叉策略矩阵**:

| 策略类型 | 组合方式 | 策略描述 | 示例 |
|---------|---------|---------|------|
| **SO策略** | 优势+机会 | 利用优势抓住机会 | 用技术领先(S)抓住新市场机会(O) |
| **WO策略** | 劣势+机会 | 克服劣势抓住机会 | 通过合作弥补资源不足(W)进入新市场(O) |
| **ST策略** | 优势+威胁 | 利用优势应对威胁 | 用品牌优势(S)抵御新进入者(T) |
| **WT策略** | 劣势+威胁 | 防御性收缩 | 退出竞争激烈的低利润市场 |

### 1.4 战略群分析 (Strategic Group Mapping)

**适用场景**: 识别竞争对手的战略定位和群组

```
                    高价格
                      │
         ┌────────────┼────────────┐
         │   高端     │   奢侈     │
         │  专业化   │   品牌     │
  高质量  │────────────┼────────────│  低质量
         │   中端     │   大众     │
         │  差异化   │   市场     │
         └────────────┼────────────┘
                      │
                    低价格
```

**绘制步骤**:
1. 选择两个关键维度(如: 价格vs质量、功能vs易用性)
2. 确定各竞品在坐标中的位置
3. 圈出战略群组
4. 识别空白市场机会

### 4.4 SWOT分析 (以某AI Agent创业公司为例)

| | **积极因素** | **消极因素** |
|---|-------------|-------------|
| **内部因素** | **S (优势)**<br>• 垂直领域深耕，行业Know-how深厚<br>• 本地化部署能力强<br>• 响应速度快，定制化灵活 | **W (劣势)**<br>• 底层技术依赖外部模型<br>• 品牌知名度低<br>• 资金实力有限 |
| **外部因素** | **O (机会)**<br>• 市场高速增长(CAGR 44%+)<br>• 企业数字化转型需求旺盛<br>• 政策支持力度大<br>• 大厂通用产品覆盖不足垂直领域 | **T (威胁)**<br>• 大厂持续入场<br>• 技术迭代快，容易落后<br>• 价格战压缩利润<br>• 监管不确定性 |

**战略建议**:
- **SO策略**: 深耕垂直行业，用本地化优势抓住企业数字化转型机会
- **WO策略**: 与基础模型厂商合作，借力大厂技术弥补自研不足
- **ST策略**: 用定制化能力构建护城河，抵御大厂标准化产品冲击
- **WT策略**: 避开与大厂正面竞争，聚焦利基市场
