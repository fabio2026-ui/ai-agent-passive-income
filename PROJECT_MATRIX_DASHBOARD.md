========================================
📈 成功标准
========================================

### 项目继续标准 (必须满足2+项)
- [ ] 月收入达到预期的50%
- [ ] 用户/客户增长趋势良好
- [ ] 单位经济模型盈利 (LTV > CAC)
- [ ] 团队执行力强
- [ ] 市场反馈积极

### 项目停止标准 (满足1项即停)
- [ ] 3个月无收入或收入<预期10%
- [ ] 获客成本 > 用户终身价值
- [ ] 市场验证失败 (无需求)
- [ ] 团队无法执行
- [ ] 外部政策/市场重大变化

### 项目放大标准 (满足3+项)
- [ ] 月收入达到或超过预期
- [ ] 获客成本 < 用户终身价值/3
- [ ] 可规模化 (不依赖个人)
- [ ] 有竞争优势/壁垒
- [ ] 市场空间足够大
