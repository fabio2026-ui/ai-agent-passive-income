## 📊 数据流向

```
我(云端AI) 
    ↓ HTTPS + X-Key认证
Tailscale Funnel
    ↓ 转发到
Mac本地控制服务:9999
    ↓ 分发命令到
├─→ 视频工厂 → 生成视频 → 上传到平台 → 变现
├─→ APP工厂 → 开发产品 → 上线销售 → 变现  
├─→ AI大脑 → 内容优化 → 提升质量 → 增加收入
└─→ 运维系统 → 自动修复 → 保证24小时运行
```
