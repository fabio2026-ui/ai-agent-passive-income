#### 任务2: Stripe注册 (10分钟)
**前置条件**: 有身份证/护照、银行卡
**步骤**:
1. 访问 https://stripe.com
2. 点击 "Start now" → 选择 "Individual"
3. 填写个人信息 (姓名、地址、电话)
4. 验证身份 (上传身份证/护照)
5. 添加银行账户 (用于收款)
6. 获取API Keys:
   - 进入 Developers → API keys
   - 复制 Publishable key 和 Secret key
   - 保存到安全地方 (我会配置到系统)

**完成后通知我**: 发送API Keys给我配置
**完成后打钩**: ☐

#### 任务3: Obsidian插件启用 (5分钟)
**前置条件**: Obsidian已安装
**步骤**:
1. 打开 Obsidian 应用

5. 点击 "打开社区插件市场"
6. 搜索并安装以下3个插件:
   - Dataview (数据分析)
   - Templater (模板)
   - Smart Connections (AI关联)
7. 逐个点击 "启用"
8. 重启 Obsidian
