# Proactive Agent Skill
# 主动智能体技能

## 用途
让AI主动监控信息、定时执行任务、主动推送提醒，不只是被动响应。

## 功能

### 1. 定时任务
- 定时执行脚本
- 周期性检查
- 定时报告
- 提醒通知

### 2. 主动监控
- 网页变化监控
- 价格变动提醒
- 内容更新检测
- 状态监控

### 3. 关键词预警
- 热点追踪
- 舆情监控
- 竞品动态
- 行业资讯

### 4. 自动推送
- Telegram推送
- 邮件通知
- 消息提醒
- 报告发送

## 使用示例

```
每天上午9点检查邮件并总结
监控竞争对手网站更新
当价格低于100元时通知我
每周一生成上周工作报告
```

## 触发方式
- 时间触发 (cron)
- 事件触发 (webhook)
- 条件触发 (if-then)
- 手动触发

## 监控源
- 网页
- API
- RSS
- 数据库
- 文件系统
