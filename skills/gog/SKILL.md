# GOG Skill - Google Workspace Integration
# Google套件集成技能

## 用途
一站式集成 Gmail、Calendar、Drive、Docs、Sheets，实现办公自动化。

## 功能

### 1. Gmail管理
- 收发邮件
- 自动分类
- 关键词提醒
- 批量操作

### 2. Calendar管理
- 自动创建日程
- 修改和同步
- 会议提醒
- 忙闲查询

### 3. Drive管理
- 文件读写
- 自动备份
- 分享管理
- 权限控制

### 4. Docs管理
- 文档创建
- 内容编辑
- 格式转换
- 协作同步

### 5. Sheets管理
- 数据录入
- 公式计算
- 图表生成
- 数据分析

## 使用示例

```
每天早上9点汇总未读邮件，标记紧急事项
把这份报告保存到云端硬盘并分享给团队
创建明天下午3点的会议，邀请张三和李四
读取Google Sheets中的销售数据并生成图表
```

## 安装要求
- Google Cloud项目
- OAuth 2.0凭证
- 启用API：Gmail, Calendar, Drive, Docs, Sheets

## 安全
- OAuth授权，不泄露密码
- Token本地存储
- 定期刷新
