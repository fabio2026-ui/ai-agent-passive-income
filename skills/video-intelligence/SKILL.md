# Video Intelligence Skill

## 用途
提取视频/文章链接的核心内容，分析方法论，并输出可执行的赚钱机会。

## 工作流程

### 输入
用户提供的链接（视频、文章、播客等）

### 处理步骤

1. **内容提取**
   - 尝试 web_fetch 获取页面内容
   - 如果失败，尝试 browser snapshot 模拟访问
   - 提取标题、正文、关键段落

2. **核心提炼**
   - 一句话总结主题
   - 提取3-5个核心观点
   - 识别可复用的方法论/框架

3. **深度学习**
   - 这个方法解决了什么问题？
   - 背后的原理是什么？
   - 有哪些变体/延伸应用？

4. **机会挖掘**
   - 这个方法能怎么赚钱？
   - 目标客户是谁？
   - 如何规模化？
   - 竞争格局如何？

5. **行动输出**
   - 立即可做的3个行动
   - 需要的资源/工具
   - 预期ROI

## 输出格式

```markdown
## 📹 内容摘要
- **来源**: [标题](链接)
- **一句话总结**: 

## 🧠 核心方法论
1. 
2. 
3. 

## 💰 赚钱机会
| 机会 | 客户 | 服务 | 难度 | 预期收益 |
|------|------|------|------|----------|

## 🚀 立即行动
- [ ] 
- [ ] 
- [ ] 

## 📚 延伸阅读
- 相关概念
- 类似案例
```

## 使用示例

用户: "https://m.toutiao.com/is/xxx"

→ 执行内容提取 → 分析 → 输出完整报告

## 注意事项
- 如果链接无法访问，提示用户复制文字或截图
- 优先提取可复用的方法论而非具体案例
- 变现分析要具体，不要泛泛而谈
- 标注信息来源和置信度
