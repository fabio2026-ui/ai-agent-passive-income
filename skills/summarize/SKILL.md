# Summarize Skill
# 智能内容摘要技能

## 用途
对长文、报告、文章、视频字幕、会议记录进行一键总结，提炼核心观点。

## 功能

### 1. 文本摘要
- 文章总结
- 报告提炼
- 论文摘要
- 新闻概览

### 2. 会议总结
- 转录文本
- 要点提取
- 行动项识别
- 决策记录

### 3. 多模式输出
- 要点列表
- 执行摘要
- 详细分析
- 思维导图结构

### 4. 批量处理
- 多文档处理
- 对比分析
- 主题聚合
- 趋势识别

## 摘要模式

| 模式 | 输出长度 | 适用场景 |
|------|----------|----------|
| bullet | 5-10条 | 快速了解要点 |
| executive | 200-300字 | 管理层汇报 |
| detailed | 500-800字 | 深度分析 |
| action-items | 动态 | 提取待办事项 |

## 使用示例

```
总结这篇技术文章
提取会议记录的核心要点
生成这份报告的执行摘要
分析这10篇文章的共同主题
```

## 支持的格式
- 纯文本
- Markdown
- PDF
- Word
- HTML
