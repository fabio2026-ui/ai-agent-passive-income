# Notion Skill
# 知识库管理技能

## 用途
AI自动读写Notion，构建个人/企业知识库，自动保存对话、笔记、文章。

## 功能

### 1. 页面管理
- 创建页面
- 编辑内容
- 页面组织
- 模板使用

### 2. 数据库操作
- 创建数据库
- 增删改查
- 筛选排序
- 关系关联

### 3. 内容同步
- 自动保存对话
- 笔记归档
- 文章收藏
- 知识整理

### 4. 协作功能
- 评论管理
- 权限控制
- 分享链接
- 版本历史

## 使用示例

```
把这次对话保存到Notion
创建今日工作总结
搜索关于"OpenClaw"的笔记
更新项目进度数据库
```

## 配置
```
NOTION_TOKEN=secret_xxx
NOTION_DATABASE_ID=xxx
```
