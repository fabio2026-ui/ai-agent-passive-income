# N8N Workflow Skill
# N8N自动化集成技能

## 用途
与N8N自动化平台集成，实现跨平台工作流自动化。

## 功能

### 1. 工作流管理
- 创建工作流
- 编辑配置
- 激活/停用
- 查看执行日志

### 2. 节点操作
- 添加节点
- 配置参数
- 连接节点
- 测试运行

### 3. 触发器
- Webhook触发
- 定时触发
- 事件触发
- 手动触发

### 4. 集成支持
- 300+应用集成
- API调用
- 数据处理
- 条件分支

## 使用示例

```
创建一个新的N8N工作流
添加Telegram通知节点
配置定时触发器
测试工作流执行
```

## 集成场景
- 数据同步
- 消息通知
- 文件处理
- 报告生成
