# GitHub Skill
# 代码仓库管理技能

## 用途
通过自然语言管理GitHub代码仓库，查看PR、Issue、提交记录，创建和管理开发任务。

## 功能

### 1. 仓库管理
- 查看仓库状态
- 浏览文件
- 搜索代码
- 查看分支

### 2. Issue管理
- 创建Issue
- 查看列表
- 添加标签
- 分配人员

### 3. Pull Request
- 查看PR列表
- 审查代码
- 合并PR
- 添加评论

### 4. 提交记录
- 查看历史
- 对比版本
- 生成提交说明
- 代码统计

### 5. Actions
- 触发工作流
- 查看状态
- 管理Secrets
- 配置环境

## 使用示例

```
查看我的OpenClaw仓库状态
创建Issue：修复登录bug
生成今天的提交总结
审查最新的Pull Request
```

## 配置
```bash
# 设置GitHub Token
claw config github --token="ghp_your_token"

# 设置默认仓库
claw config github --default-repo="username/repo"
```

## 安全
- 使用Personal Access Token
- 最小权限原则
- Token本地存储
