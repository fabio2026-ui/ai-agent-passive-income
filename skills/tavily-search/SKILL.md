# Tavily Search Skill
# AI驱动的智能搜索技能

## 用途
专为AI Agent设计的搜索平台，支持网页、新闻、财经搜索，提供结构化信息获取能力。

## 功能

### 1. 网页搜索
- 实时搜索结果
- 内容摘要
- 来源引用
- 相关性排序

### 2. 新闻搜索
- 最新新闻
- 热点追踪
- 行业资讯
- 趋势分析

### 3. 财经搜索
- 股票信息
- 市场数据
- 财报分析
- 投资建议

### 4. 深度研究
- 多源聚合
- 观点对比
- 数据验证
- 报告生成

## 使用示例

```
搜索"OpenClaw最新功能"
查找"AI行业2026年趋势"
研究"特斯拉最新财报"
获取"Python最佳实践"
```

## 优势
- 为AI优化的搜索结果
- 结构化输出
- 高准确性
- 实时更新

## API集成
```python
from tavily import TavilyClient

client = TavilyClient(api_key="your-api-key")
result = client.search("query", search_depth="advanced")
```
