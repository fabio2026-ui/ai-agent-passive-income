# Capability Evolver Skill
# AI能力自动进化技能

## 用途
让AI Agent从错误、反馈、纠正中自动学习，持续提升回答质量和执行效率。

## 功能

### 1. 错误分析
- 自动记录错误案例
- 分类错误类型
- 分析根本原因
- 提取改进点

### 2. 反馈固化
- 将用户反馈转为记忆
- 识别高频需求
- 优化响应模式
- 个性化调整

### 3. 策略优化
- 自动调整Prompt
- 优化执行路径
- 改进决策逻辑
- 提升成功率

### 4. 持续学习
- 每日自我审查
- 模式识别
- 知识更新
- 能力扩展

## 工作原理

```
1. 记录每次交互
2. 分析成功/失败案例
3. 识别改进机会
4. 自动调整策略
5. 验证改进效果
6. 固化到长期记忆
```

## 使用示例

```
启用能力进化模式
分析最近100次交互的改进点
根据用户反馈优化回答风格
自动学习新技能的使用方法
```

## 配置

```yaml
evolution:
  enabled: true
  review_interval: 24h
  learning_rate: adaptive
  memory_depth: long_term
  
feedback:
  auto_collect: true
  user_confirmation: false
  improvement_threshold: 0.8
```

- 越用越准确
- 越用越贴合用户习惯
- 自动适应新场景
- 持续提升效率
