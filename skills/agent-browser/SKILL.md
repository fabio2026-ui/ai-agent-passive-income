# Agent Browser Skill
# 浏览器自动化技能

## 用途
让AI能够自动控制浏览器，进行网页操作、数据采集、表单填写等。

## 功能

### 1. 网页导航
- 自动打开URL
- 页面等待和刷新
- 多标签管理
- 历史记录

### 2. 元素交互
- 点击按钮
- 填写表单
- 选择下拉框
- 上传文件

### 3. 数据提取
- 抓取文本内容
- 提取表格数据
- 下载图片/文件
- 解析JSON

### 4. 截图和PDF
- 页面截图
- 元素截图
- 生成PDF
- 全页截图

### 5. 自动化流程
- 录制操作序列
- 回放自动化
- 条件判断
- 循环处理

## 使用示例

```
打开example.com并登录
搜索"OpenClaw"并抓取前10个结果
填写表单并提交
每天定时检查网页更新
```

## 技术实现
- 基于Playwright
- 支持Chrome/Firefox/WebKit
- 无头模式运行
- 支持代理和Cookie

## 安全注意事项
- 不在公共环境输入密码
- 敏感操作需二次确认
- 定期清理Cookie
