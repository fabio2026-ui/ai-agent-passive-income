# Mission Control Skill
# 任务简报聚合技能

## 用途
将任务管理、日历事件和通知消息聚合为一份早间简报，个人效率的"控制塔"。

## 功能

### 1. 任务聚合
- 汇总待办事项
- 截止日期提醒
- 优先级排序
- 完成进度

### 2. 日历概览
- 当日日程
- 次日预览
- 会议提醒
- 冲突检测

### 3. 通知摘要
- 重要邮件
- 消息提醒
- 系统通知
- 关键词过滤

### 4. 简报生成
- 智能摘要
- 行动建议
- 时间估算
- 自动推送

## 使用示例

```
每天早上8点生成今日简报
汇总所有待办和会议
提醒我今天的重点任务
推送摘要到Telegram
```

## 配置

```yaml
brief_time: "08:00"
timezone: "Asia/Shanghai"
sources:
  - todoist
  - notion
  - google_calendar
  - slack
  - email
```

## 输出格式

```markdown
# 📋 今日简报 - 2026-04-01

## ⏰ 日程 (3个)
- 09:00 团队站会
- 14:00 项目评审
- 16:00 客户会议

## ✅ 待办 (5个)
- [P0] 完成API文档
- [P1] 回复客户邮件
- [P2] 更新项目进度

## 🔔 提醒
- 2封未读邮件（1封紧急）
- Slack 3个@提及

## 💡 建议
今天会议较多，建议提前准备材料
```
