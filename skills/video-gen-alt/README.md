## 🚀 快速开始

### 1. 获取即梦Session ID

1. 访问 https://jimeng.jianying.com/ 注册/登录
2. F12 打开开发者工具 → Application → Cookies
3. 复制 `sessionid` 值

### 2. 设置环境变量

```bash
export JIMENG_SESSION_ID='your_session_id_here'
```

### 3. 生成视频

```bash
# 单视频生成
python generate_video.py "富二代骑着电动车穿梭在城市夜景中，霓虹闪烁的摩天大楼，赛博朋克风格"

# 批量生成
python batch_generate.py
```
