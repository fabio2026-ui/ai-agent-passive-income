## 📝 使用方法

### 第一步：获取Session ID
```bash
# 1. 访问 https://jimeng.jianying.com/ 登录
# 2. F12 → Application → Cookies → 复制 sessionid
# 3. 设置环境变量
export JIMENG_SESSION_ID='your_session_id_here'
```

### 第二步：生成视频
```bash
# 单视频生成
cd /root/.openclaw/workspace/skills/video-gen-alt
python3 generate_video.py "富二代骑着电动车穿梭在城市夜景中，霓虹闪烁的摩天大楼，赛博朋克风格"

# 批量生成
python3 batch_generate.py
```
