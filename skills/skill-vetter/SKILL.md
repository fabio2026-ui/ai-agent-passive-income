# Skill Vetter
# 技能安全审计技能

## 用途
在安装技能前自动扫描安全风险，防范恶意代码、密钥窃取、数据泄露。

## 背景
2026年初爆发ClawHavoc事件，超过341个恶意技能被上传，Skill Vetter成为必装防护工具。

## 功能

### 1. 源码检查
- 静态代码分析
- 依赖检查
- 漏洞扫描
- 后门检测

### 2. 权限评估
- 权限分析
- 过度权限检测
- 敏感操作识别
- 风险评级

### 3. 行为监控
- 网络请求分析
- 文件系统监控
- 环境变量检查
- 可疑行为识别

### 4. 安全报告
- 风险评分
- 详细报告
- 修复建议
- 白名单管理

## 使用示例

```bash
# 审计技能
skill-vetter audit 技能名称

# 批量审计
skill-vetter audit-batch 技能列表

# 查看报告
skill-vetter report 技能名称
```

## 检测内容
- 恶意代码注入
- 密钥窃取行为
- 数据上传外泄
- Cookie窃取
- 系统命令执行
- 代码混淆检测

## 安全等级
- 🟢 安全 - 可放心安装
- 🟡 警告 - 需谨慎使用
- 🔴 危险 - 不建议安装
