## 🚀 快速发布步骤（5分钟）

### Step 1: 注册开发者账号
1. 访问 https://chrome.google.com/webstore/devconsole
2. 登录你的Google账号
3. 支付 **$5一次性费用**
4. 等待审核（通常即时）

### Step 2: 上传扩展
1. 点击 "New Item"
2. 上传文件: `chrome-extension-v1.0.0.zip`
3. 等待自动解析

### Step 3: 填写商店信息

**必填字段**（已准备内容）：

| 字段 | 内容 |
|------|------|
| **Description** | Quick access to 50+ cybersecurity articles, tools, and best practices. Perfect for developers, security engineers, and IT professionals. Features: 🔒 50+ curated articles, 🛠️ Security tools directory, 📊 Quick reference cards, 🔍 Search functionality |
| **Category** | Developer Tools |
| **Language** | English |
| **Website** | https://fabio2026-ui.github.io/ai-agent-passive-income |
| **Support Email** | （填你的邮箱）|

### Step 4: 截图和宣传图

**需要准备**（可用现有网站截图）：
- 1-5张扩展截图 (1280x800 或 640x400)
- 可选: 宣传图 (1400x560)

**快速方案**：用网站 dashboard.html 截图

### Step 5: 定价和发布
- **定价**: Free
- **Visibility**: Public
- **Publish**: 点击 "Submit for Review"
