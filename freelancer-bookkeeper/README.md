# Freelancer Bookkeeper - MVP Spec

## 目标
3天MVP，自由职业者记账自动化工具

## 核心功能
1. 收据上传 (图片/PDF)
2. AI自动分类 (DeepSeek API)
3. 简单仪表盘 (收入/支出/利润)
4. 税季报表导出
5. Stripe订阅支付

## 技术栈
- Frontend: Vanilla JS + Tailwind
- Backend: Cloudflare Workers
- AI: DeepSeek API
- DB: Cloudflare D1
- Auth: JWT
- Payment: Stripe

## 定价
- Free: 20笔/月
- Pro: €29/月，无限笔数

## 目标用户
自由职业者、独立开发者、咨询师

## 竞品
- QuickBooks (过度复杂，€30+/月)
- FreshBooks (同样复杂)
- Wave (免费但功能弱)

## 差异化
- 极简界面
- AI自动分类
- 专为自由职业者优化
- 价格更便宜
