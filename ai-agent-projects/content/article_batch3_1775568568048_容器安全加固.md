# Kubernetes容器安全加固指南

## 简介
容器化带来新安全挑战。本文详解K8s安全加固。

## 镜像安全

### 最小化镜像
- 使用Alpine/Distroless
- 移除包管理器
- 多阶段构建

```dockerfile
# 多阶段构建示例
FROM node:16 AS builder
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production

FROM node:16-alpine
COPY --from=builder /app .
USER node
CMD ["node", "index.js"]
```

### 镜像扫描
- Trivy
- Snyk
- Clair

## 运行时安全

### Pod安全
```yaml
securityContext:
  runAsNonRoot: true
  readOnlyRootFilesystem: true
  allowPrivilegeEscalation: false
  capabilities:
    drop:
      - ALL
```
