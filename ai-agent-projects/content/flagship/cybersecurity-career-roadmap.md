# 🔒 网络安全职业路径完整指南

## 职业阶段路线图

### 🌱 阶段一：入门 (0-1年)
**目标职位**: Security Analyst, SOC Analyst, Junior Penetration Tester

**核心技能**:
- 网络基础 (TCP/IP, DNS, HTTP)
- Linux/Windows系统管理
- 基础编程 (Python, Bash)
- 安全工具使用 (Wireshark, Nmap, Metasploit)

**认证推荐**:
| 认证 | 难度 | 费用 | 价值 |
|------|------|------|------|
| CompTIA Security+ | ⭐⭐ | $370 | ⭐⭐⭐⭐⭐ |
| Google Cybersecurity Certificate | ⭐ | $39/月 | ⭐⭐⭐⭐ |
| eJPT (eLearnSecurity) | ⭐⭐ | $200 | ⭐⭐⭐⭐ |

**学习资源**:
- TryHackMe (初学者友好)
- Hack The Box (进阶练习)
- Cybrary (免费课程)

**预期薪资**: $50,000 - $70,000

---

### 🚀 阶段二：中级 (1-3年)
**目标职位**: Security Engineer, Senior Analyst, Consultant

**核心技能**:
- 渗透测试方法论
- SIEM工具 (Splunk, ELK, QRadar)
- 云安全 (AWS/Azure/GCP)
- 安全架构设计基础

**认证推荐**:
| 认证 | 难度 | 费用 | 价值 |
|------|------|------|------|
| OSCP | ⭐⭐⭐⭐ | $1,600 | ⭐⭐⭐⭐⭐ |
| CISSP | ⭐⭐⭐ | $749 | ⭐⭐⭐⭐⭐ |
| AWS Security Specialty | ⭐⭐⭐ | $300 | ⭐⭐⭐⭐ |

**项目建议**:
- 搭建家庭实验室
- 参与CTF比赛
- 贡献开源安全工具
- 写技术博客

**预期薪资**: $80,000 - $120,000

---

### 🏆 阶段三：高级 (3-5年)
**目标职位**: Senior Security Engineer, Security Architect, Team Lead

**核心技能**:
- 安全架构设计
- 威胁建模 (STRIDE, DREAD)
- 合规框架 (SOC2, ISO 27001, NIST)
- 团队管理

**认证推荐**:
| 认证 | 难度 | 费用 | 价值 |
|------|------|------|------|
| CISSP | ⭐⭐⭐ | $749 | ⭐⭐⭐⭐⭐ |
| CISM | ⭐⭐⭐ | $760 | ⭐⭐⭐⭐ |
| CRISC | ⭐⭐⭐ | $760 | ⭐⭐⭐⭐ |

**预期薪资**: $130,000 - $180,000

---

### 👑 阶段四：专家 (5年+)
**目标职位**: CISO, Security Director, Principal Architect

**核心技能**:
- 战略规划
- 预算管理
- 跨部门协作
- 行业影响力

**发展路径**:
- 管理路线: Team Lead → Manager → Director → CISO
- 技术路线: Senior → Staff → Principal → Distinguished Engineer

**预期薪资**: $200,000 - $500,000+

## 专业技能树

### 🛡️ 防御方向 (Blue Team)
```
SOC Analyst
    ↓
Security Engineer
    ↓
Security Architect
    ↓
CISO
```

**关键技能**:
- SIEM/SOAR
- 威胁情报
- 事件响应
- 安全运营

---

### ⚔️ 攻击方向 (Red Team)
```
Junior Pentester
    ↓
Penetration Tester
    ↓
Red Team Operator
    ↓
Red Team Lead
```

**关键技能**:
- 漏洞挖掘
- 社会工程学
- 物理安全测试
- 高级持续威胁 (APT) 模拟

---

### 🔧 安全开发 (DevSecOps)
```
Security Developer
    ↓
AppSec Engineer
    ↓
DevSecOps Lead
    ↓
Security Architect
```

**关键技能**:
- 安全编码
- SAST/DAST
- CI/CD安全
- 容器安全
